## 第一部分：王朝奠基与神话确立期
*时期：黑蔑特第一王朝初期（马塔肖萨门一世统治期）*

这一时期是图斯克文明从部落城邦向统一帝国转型的关键节点，也是“海典”神话体系的确立时代。

### 1. <b><ruby>汉布瓦西亚<rt>Hambwasia</rt></ruby></b>家族
- **【家族词源】**：意为“神话”或“史诗”，传统史官家族 [^2]。
- **【代表人物】**：<b><ruby>汉布瓦西亚·索里亚兹<rt>Hambwasia Soryaz</rt></ruby></b>
- **【头衔】**：首任宫廷修辞官 / 《诸神篇》主笔
- **【生平与功绩】**：
    索里亚兹被视为王朝合法性的神学奠基人。在马塔肖萨门一世统一战争期间，他着手将各城邦混乱的部落传说整合进统一的“六主神体系”。针对抵抗激烈的“佩尔南”与“格那拉”城邦，索里亚兹创造性地将其保护神降格为“下三神” [^1]，从而在神学层面瓦解了敌对城邦的抵抗意志。史料记载，他曾于海边悬崖枯坐九昼夜，声称在织海神的幻象中获得了构建神学秩序的灵感。

### 2. <b><ruby>铁苏库<rt>Tyesuku</rt></ruby></b>家族
- **【家族词源】**：意为“暖石”，源自早期铁匠职业，后成为西部山脉之名 [^2]。
- **【代表人物】**：<b><ruby>铁苏库·特罗列格<rt>Tyesuku Troleug</rt></ruby></b>
- **【头衔】**：首席军械官 / 铁苏库山脉领主
- **【生平与功绩】**：
    作为帝国西部的屏障，特罗列格在对抗“恐渎念”海盗的战争中居功至伟 [^1]。他改良了冶炼技术，发现加入特定比例“骨炭”可锻造出更坚硬的钢材（图斯克钢雏形）。因其卓越贡献，马塔肖萨门一世将西部富矿山脉封赐予该家族。特罗列格性格暴烈，据传曾因刀剑质量问题，亲自将负责工匠投入海中“献祭翻海神”。

### 3. <b><ruby>洛萨安<rt>Lousan</rt></ruby></b>家族
- **【家族词源】**：意为“正道”，帝国道路工程世家 [^2]。
- **【代表人物】**：<b><ruby>洛萨安·耶普索伊<rt>Lousan Jeposoi</rt></ruby></b>
- **【头衔】**：帝国巡路使 / 皇家信使总管
- **【生平与功绩】**：
    耶普索伊提出了“道通神达”的理念，主持修建了连接首都黑蔑特与南部图斯克盆地的第一条直道——马塔肖萨门大道。针对盆地内复杂的沼泽地形，他首创“浮土沉石”筑路法。其家族格言<b>“行直道，守正心”</b>后演变为帝国官僚阶层的座右铭。

### 4. <b><ruby>伊勒赛特<rt>Ilesait</rt></ruby></b>家族
- **【家族词源】**：意为“辉煌”，古代著名的女祭司家族 [^2]。
- **【代表人物】**：<b><ruby>伊勒赛特·宁斯威<rt>Ilesait Ningswi</rt></ruby></b>
- **【头衔】**：织海神大祭司 / 圣波拉斯蓬的启蒙导师
- **【生平与功绩】**：
    在尚武的王朝初期，宁斯威代表了神权的柔性力量，负责主持春季“开海仪式”。她是圣徒波拉斯蓬的伯乐，最早发掘了其眼中的“虚实裂隙”之力 [^1]。作为极少数能解读“后十戒（隐秘之律）”的高阶神职人员，宁斯威晚年为封印海中异物，自愿执行“藏海”仪式走向深渊，不知所踪。

### 5. <b><ruby>吉瓦泰斯<rt>Giateis</rt></ruby></b>家族
- **【家族词源】**：意为“秘识”，象征掌握隐秘知识 [^2]。
- **【代表人物】**：<b><ruby>吉瓦泰斯·波特伊斯克<rt>Giateis Porteiske</rt></ruby></b>
- **【头衔】**：宫廷秘术顾问 / 异端审判者
- **【生平与功绩】**：
    不同于汉布瓦西亚家族的光明叙事，吉瓦泰斯家族负责清理阴影中的威胁。波特伊斯克主导了对前朝及被征服部落“邪典”的搜集与销毁工作。他是“马塔肖萨门戒律”中“后十戒”的主要起草者，界定了凡人不可触碰的灵语禁忌。在当时的贵族圈中，其名讳具有极强的震慑力。

## 第二部分：帝国扩张与职能完善期
*时期：第一王朝扩张期及制度建设期*

随着疆域扩大，帝国在情报、边防、建筑与医疗领域涌现了一批专业职能家族。

### 1. <b><ruby>赫图克<rt>Hetuke</rt></ruby></b>家族
- **【家族词源】**：意为“夜行者”（*hetu* 夜 + *ke* 者） [^2]。
- **【代表人物】**：<b><ruby>赫图克·韦腾<rt>Hetuke Weten</rt></ruby></b>
- **【头衔】**：哈希特暗卫统领 / “静默署”长官
- **【生平与功绩】**：
    出身于哈希特贫民窟更夫行会。韦腾被皇帝收编后，建立了一张覆盖首都各阶层的情报网。其最著名战绩是在“血色满月”之夜提前破获旧贵族焚粮阴谋。家族格言<b>“于无声处听惊雷”</b>后成为帝国情报机构信条。

### 2. <b><ruby>萨梅姆格雷克<rt>Samemgreke</rt></ruby></b>家族
- **【家族词源】**：意为“山民”或“守山人” [^2]。
- **【代表人物】**：<b><ruby>萨梅姆格雷克·古盾<rt>Samemgreke Gudun</rt></ruby></b>
- **【头衔】**：希娜支镇守备官 / 北方军团指挥官
- **【生平与功绩】**：
    该家族原为被招安的山地部落。在主力南征之际，古盾率八百“守山人”在希娜支镇死守一月，抵挡了俺东蛮族三千人的围攻。战后获封世袭伯爵，该家族从此被称为“帝国的北墙”。

### 3. <b><ruby>戈劳盾<rt>Goiraudun</rt></ruby></b>家族
- **【家族词源】**：意为“宏殿”（*goirau* 殿 + *dun* 大） [^2]。
- **【代表人物】**：<b><ruby>戈劳盾·奥勒曼<rt>Goiraudun Oleman</rt></ruby></b>
- **【头衔】**：皇家首席营造师 / 博尔拉索姆港总设计师
- **【生平与功绩】**：
    奥勒曼是当时杰出的石匠与数学家，主持了博尔拉索姆港扩建工程，并发明“水下沉箱法”。他重新设计的哈希特皇宫防御体系，完美结合了美学与军事防御功能。

### 4. <b><ruby>努昔卜<rt>Numsib</rt></ruby></b>家族
- **【家族词源】**：意为“千叶”（*numrea* 千 + *sib* 叶），象征繁荣与变数 [^2]。
- **【代表人物】**：<b><ruby>努昔卜·德奥德<rt>Numsib Dayaud</rt></ruby></b>
- **【头衔】**：宫廷御医 / 皇家植物园管理者
- **【生平与功绩】**：
    针对南征军队遭遇的热病，德奥德编纂了首部《帝国草本志》，并发明“千叶草”熏蒸法，大幅降低了非战斗减员。因掌握剧毒草药知识，该家族在宫廷中被称为“握着生与死之手的人”。

## 第三部分：行政改革与共和时期
*时期：航海扩张、行政改革及后期的共和共治时代*

### 1. <b><ruby>波拉斯索姆<rt>Porlasom</rt></ruby></b>家族
- **【家族词源】**：意为“古直”或“守旧” [^1]。
- **【代表人物】**：<b><ruby>波拉斯索姆·卓奥西<rt>Porlasom Drodaucy</rt></ruby></b>
- **【头衔】**：首席大法官 / “军镇法”制定者
- **【生平与功绩】**：
    活跃于坎索伊改革时期。卓奥西协助皇帝制定严苛的《军镇律》，并以“法不阿贵”著称。曾顶住皇室压力，宣判违律杀降的皇侄死刑。该家族被誉为“帝国的直尺”。

### 2. <b><ruby>巴罗尔<rt>Barol</rt></ruby></b>家族
- **【家族词源】**：意为“深水” [^1]。
- **【代表人物】**：<b><ruby>巴罗尔·瓦拉<rt>Barol Vala</rt></ruby></b>
- **【头衔】**：皇家舰队领航长 / 诺瓦亚洋流图绘制者
- **【生平与功绩】**：
    出身海边采珠人世家，拥有感知水流变化的异能。瓦拉发现了隐秘的“黑水洋流”，助推图斯克舰队在诺瓦亚海的扩张。晚年绘制《诺瓦亚水文志》，被尊为“洋流之母”。

### 3. <b><ruby>古督克<rt>Guduke</rt></ruby></b>家族
- **【家族词源】**：意为“缝补者”或“储藏者” [^1]。
- **【代表人物】**：<b><ruby>古督克·安赛特<rt>Guduke Ansait</rt></ruby></b>
- **【头衔】**：大司农 / 战时后勤总管
- **【生平与功绩】**：
    在阿贵吉奥发动的“瑞联地绞肉战”期间，安赛特通过“分段转运法”和密封肉干技术，维持了脆弱的补给线。虽因重税被民间记恨，但被官方评价为“比十个军团更重要的人”。

### 4. <b><ruby>瑞兰索姆<rt>Ruilansom</rt></ruby></b>家族
- **【家族词源】**：意为“多财者” [^1]。
- **【代表人物】**：<b><ruby>瑞兰索姆·因古霍<rt>Ruilansom Yingukho</rt></ruby></b>
- **【头衔】**：首任瑞联地总督 / 皇家铸币厂监督
- **【生平与功绩】**：
    因资助波拉斯蓬起家，后建立了瑞联地矿奴制度，为王朝极盛期提供了财政支撑。他是首位建议确立“古冶”（银币）为标准货币的官员。共和时期，该家族成为元老院寡头之一。

## 第四部分：衍生姓氏与分支家族考
*注：本部分收录带有后缀 <b>-norko</b>（意为“子嗣”或“后代”）的特殊姓氏，此类姓氏多用于表示庶出分支、流亡贵族或归化异族。*

### 1. <b><ruby>海姆诺科<rt>Heimnorko</rt></ruby></b> —— “流放的日之子”
- **【词源】**：*Heim* (黑蔑特/太阳) + *norko* (子嗣)
- **【考据】**：源自黑蔑特皇室旁系。卓奥西一世终结“七家共治”时，将皇室反对派流放至南方边境。该支系后成为南方拓荒领主，家徽为“破碎的太阳”。

### 2. <b><ruby>奎诺科<rt>Kuinorko</rt></ruby></b> —— “黄金的遗孤”
- **【词源】**：*Kui* (奎列奥/黄金) + *norko* (子嗣)
- **【考据】**：源自前朝奎列奥王室残余。王朝更迭后，部分旧王族宣誓效忠并改姓，转型为哈希特城的银行家与巨商。

### 3. <b><ruby>莫斯诺科<rt>Mosnorko</rt></ruby></b> —— “定居的骑手”
- **【词源】**：*Mos* (莫斯瓦里克/骑) + *norko* (子嗣)
- **【考据】**：源自北方咕洛族莫斯瓦里克氏族。随着民族融合政策推行，定居城市的咕洛后裔简化了姓氏，多在帝国陆军中担任骑兵军官。

### 4. <b><ruby>铁苏诺科<rt>Tyesunorko</rt></ruby></b> —— “暖石的工匠”
- **【词源】**：*Tyesu* (铁苏库/暖) + *norko* (子嗣)
- **【考据】**：源自铁苏库家族的庶出分支。在铁苏库第二王朝集权改革中分家，该支系回归工匠本源，控制着帝国西部的大型兵工厂。

### 5. <b><ruby>维斯诺科<rt>Weisnorko</rt></ruby></b> —— “林荫下的幸存者”
- **【词源】**：*Weis* (维斯尤丰) + *norko* (子嗣)；双关语 *Wise* (树)。
- **【考据】**：源自被清洗的外戚维斯尤丰家族。
    - **历史背景**：阿贵吉奥发动“铁血摄政”清洗时，该家族幸存者逃入安纽科萨丛林。
    - **代表人物**：<b><ruby>维斯诺科·莱斯维斯<rt>Weisnorko Laiswise</rt></ruby></b>，西部林业行会总长。
    - **现状**：表面为木材大亨，实则在地下图书馆收集散佚的皇家档案，试图还原家族被清洗的历史真相。
