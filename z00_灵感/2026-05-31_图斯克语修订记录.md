|------|------|

| 3.4.1.1.2 | 基础代词：ia/sou/mbo 单数五格缩合 + 复数四格（iakit/soukit/mbokit） |
| 3.4.1.1.3 | 尊称代词：hirin/sorin 斜格缩合六格 + mibok 类推五格 + 复数 hiki/siki/miboki |
| 3.4.1.1.4 | 特称：Sekui/Kuisou 不变格 + Kuisou 迁说法三例句 |
| 3.4.1.1.5 | 谦称：wan/wak/haam/sewank/sewam 格位表 + wanti/hamti 复数 |
| 3.4.1.1.6 | 区域尊称：omsou 半变格、lakome 名词变格 |

**核心设计决策**：
- 基础代词走显指路径（`-em` 系），语法化为唯一格形式，不区分定指/显指

- mibok 主格与与格同形，口语借用咕洛语位格 `-met` 消歧
- 谦称默认走定指路径（`-im`/`-il` 等），显指为有标形式
