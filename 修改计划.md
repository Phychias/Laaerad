# 拉埃拉德世界重构修改计划

> 本文档记录 2026 年 6 月讨论确定的拉埃拉德世界重构方案——将**图斯克帝国**重塑为**拜占庭式文明**。所有连接代表**主要政权的转移**。

---

## 一、核心定位

**图斯克帝国 ≈ 拜占庭帝国**

| 拜占庭 | 图斯克 |
| :--- | :--- |
| 安纳托利亚 + 君士坦丁堡 | 图斯克盆地 + 哈希特城 |
| 博斯普鲁斯海峡 | 黑蔑特海峡 |
| 地中海 | 诺瓦亚海 |
| 罗马法 + 希腊文化 | 奎列奥法 + 图斯克文化 |
| 基督教 | 新兴一神教 |
| 阿拉伯 + 突厥入侵 | **卡扎卡 + 俺东 + 咕洛三线入侵** |

## 五、待新建概念

### 卡扎卡人（类似阿拉伯/萨珊波斯）

- 起源地：南方（古督卡走廊以南）
- 一神教：具体教派待定（类似伊斯兰/犹太教）
- 扩张性强：与图斯克长期对峙
- 文化特征：法律严明、商业发达、军事扩张
- 待补充：起源、一神教义、与图斯克关系

### 新兴一神教（类似基督教/伊斯兰教）

- 创立者：待定（Q5）
- 与海神教关系：替代/融合/斗争
- 传播路径：从卡扎卡 → 图斯克 → 周边
- 在图斯克地位：黑蔑特第二时期应已普及

## 六、待澄清问题（执行前必须拍板）

### Q1：巴西卜王朝的性质

- **(a) 核心本土反拉普里奥起义**（图表暗示：C→D 表示政权从诺瓦亚转移到巴西卜，巴西卜在古图斯克诸城子图）
- (b) 拉普里奥-卡扎卡过渡政权
- (c) 拉普里奥王室延续

### Q2：伯帖斯抓护与莫斯瓦里克的关系

- **(a) 莫斯瓦里克内部分裂产物**（F⇢L 暗示）
- (b) 莫斯瓦里克残余 + 卡扎卡元素混合
- (c) 完全独立的咕洛新支系

### Q3：终局"图斯克帝国"形成时间

- (a) 铁苏库第二时期的改革
- (b) 铁苏库第一已开始，伯帖斯抓护 + 铁苏库第二完成
- (c) 伯帖斯抓护已开始建立，铁苏库第二才稳定

### Q4：卡扎卡人的具体定位

- (a) 卡扎卡 = 拉普里奥的同源或同盟
- (b) 卡扎卡 = 独立的内陆型一神教政权（类似萨珊波斯）
- (c) 卡扎卡 = 一神教意识形态的载体

### Q5：一神教的起源

- (a) 卡扎卡人创立（类似伊斯兰教）
- (b) 独立起源，被多个族群采纳（类似基督教）
- (c) 图斯克内部宗教改革

### Q6：拉普里奥的实际形态

- (a) 拉普里奥 = 海权政权，未必攻陷哈希特
- (b) 拉普里奥确实攻陷，但本质是海上帝国
- (c) 拉普里奥仅控制诺瓦亚海岛屿与海岸
