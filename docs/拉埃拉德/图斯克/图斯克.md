# <ruby>图斯克<rt>trusk</rt></ruby>

图斯克一词在拉埃拉德世界中有许多释义。

## 词源

“图斯克”一词来源于海峡图斯克语中的“trusk”，而“trusk”则源于更古老的[原始语](拉埃拉德原始语.md) ——“tʰro”。“tʰro”意为“制造”，这一称呼可能源自于早期图斯克人发达的造船业和锻造水平。
