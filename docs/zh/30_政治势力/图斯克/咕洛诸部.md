# <ruby>咕洛<rt>Glömer</rt></ruby>诸部

!!! wiki "词条信息"
    | 属性 | 描述 |
    | :--- | :--- |
    | **名称** | <ruby>咕洛<rt>Glömer</rt></ruby>诸部 |
    | **地理范围** | 黑蔑特海峡以东，赵黠斯与安纽科萨森林以北 |
    | **主要语言** | <ruby>咕洛语<rt>Glömer</rt></ruby>（图斯克语称 *Gulomer*） |
    | **政治体制** | 部落联盟 / 宗族家长制（后期受图斯克影响集权化） |
    | **主要信仰** | 祖先崇拜、<ruby>灵语<rt>Johuz</rt></ruby>体系 |
    | **经济模式** | 游牧、渔猎、少量耐寒农业 |


**咕洛诸部**是指分布于黑蔑特海峡以东、赵黠斯与安纽科萨广袤森林以北地区的多个游牧及渔猎部族统称。尽管因居住地气候与物产条件的差异，各部落在习俗、政治结构及生计方式上存在较大区别，但他们共享着相似的起源传说与语言体系，是图斯克帝国东北边境的重要组成部分。

## 概念与词源 📜

<ruby>咕洛<rt>Glömer</rt></ruby>这一称呼，在咕洛语中的本意为“河边”。

- **对外称呼的由来**：最初，图斯克人将该词用于指代他们最早接触的、自称为“Glömer”的一支氏族。随着时间推移，这个名称在图斯克语中固化，成为对北方所有相关民族的统称。
- **自我认同的演变**：经过数百年的交流与融合，尤其是原本的“Glömer”支系逐渐被同化或消亡后，其他诸部也逐渐接受并内化了这一族群称谓。

### 语言差异
<ruby>咕洛<rt>Glömer</rt></ruby>一词在母语中包含与 *o* 对立的前元音 *ö*。由于图斯克语中缺乏此音，因此常被转写为 *Gulomer*。二者之间的语音对应关系成因尚无定论，学界普遍认为这是双方接触初期因语音认知差异而产生的固化现象。

## 地理环境

咕洛族的领地位于北海与东海之间的狭长走廊地带，地势北高南低，地形复杂多样。

### 地形与气候
- **东部（咕洛草原）**：
    - **北部**：地势较高，水草丰美，是优良的牧场。
    - **南部**：受强烈南下西风带及北海暖流影响，虽然温暖但极易发生洪涝与风暴灾害。
- **西部（米斯特那提拉山脉）**：
    - **北部山区**：纬度较高，呈现雪原、高山针叶林、苔原等混合生态群系。
    - **南部山区**：受暖流调节，生长有较多温带针阔混交林。

![trusk_eco](trusk_eco.png)

### 生态与生物
- **植被**：以广阔的草原为主，涵盖从河谷草原到高山草甸的多种形态。高海拔或高纬度地区分布有耐寒的苔原植被（苔藓、地衣、矮灌木）。
- **动物**：
    - **家畜**：**牛**、**羊**是食物与经济基石；**马**主要用于交通与军事活动。
    - **野生动物**：**狼群**是主要的陆地捕食者；**鹰**占据天空生态位；**狐狸**与**野犬**广泛分布于各类生境中。

## 历史沿革

### 起源传说
<ruby>咕洛<rt>Glömer</rt></ruby>族的早期历史因游牧特性缺乏文字记载而不可考。根据[咕洛传说](../../50_语言文化/宗教与传说/咕洛传说.md)，咕洛族人是由女神**拉姆若**以自身血肉所化，并在神山**卡咯达**的庇佑下幸存于大灾变时期。

### <ruby>莫斯瓦里克<rt>Moswarike</rt></ruby>王朝的建立
咕洛族的首个“大一统”时期被称为**莫斯瓦里克王朝**（意为马族王朝），由传奇领袖<ruby>顾卓八顾<rt>Guuchobagu</rt></ruby>建立。

- **崛起**：<ruby>顾卓八顾<rt>Guuchobagu</rt></ruby>原为在草原斗争中失败沦为奴隶的幸存者。他利用宗教迷信与个人灵力，在寒冬中团结了各部族的底层奴隶。
- **夺权**：在一次大部族集会上，他以“拉卓达之名”发动政变，清洗了醉酒的长老阶层。通过许诺财富（牛羊）与解放奴隶，他成功整合了军队与主要部族。
- **统治**：他自封为<b><ruby>萨劼皮奥马克<rt>Sajepyomak</rt></ruby></b>（天下大长老）。尽管此时的统一具有较强的名义性质，实际权力仍受制于大长老会及地方氏族，但他成功唤醒了咕洛人的民族自觉。

### 继承危机与南逃
顾卓八顾死后无嗣，其弟<ruby>欧顾卓特洛<rt>Onguchotro</rt></ruby>继位。欧顾卓特洛逝世后，大长老会拒绝承认其子<ruby>欧顾卓特洛诺<rt>Onguchotronor</rt></ruby>的继承权，转而拥立旁系。欧顾卓特洛诺反抗失败，被迫南逃至赵黠斯。

### 王朝复辟与改革
流亡期间，欧顾卓特洛诺借助忠诚的边境部族及从图斯克人处获取的先进装备，发动反攻并击败大长老会，确立了莫斯瓦里克家族的神圣统治地位。

**主要改革措施**：
1.  **政治集权**：借鉴图斯克制度，削弱大长老会实权，将其降级为咨询议事机构。
2.  **魔法复兴**：建立<b><ruby>皎忽匝依<rt>Johuzay</rt></ruby></b>（祭司）制度。利用当时“快照”原理较远的有利时机，建立了成熟的“灵语”魔法体系。
3.  **文化反超**：由于同期图斯克陷入与拉普里奥人的战乱导致典籍损毁，咕洛族的灵语造诣一度反超南方，为日后在图斯克帝国西境建立咕洛王朝奠定了基础。

## 政治与社会体制 ⚖️

### 基础结构
由于游牧特性，咕洛族的政治单位主要基于地理分布与血缘关系，结构相对松散。

- **<b><ruby>氏族<rt>Krota</rt></ruby></b>**：基础政治单位（如山地氏族、河流氏族）。
- **<b><ruby>家族<rt>Shnomca</rt></ruby></b>**：社会基本细胞，实行家长制，家族成员共同拥有牲畜与帐篷。
- **<b><ruby>长老会<rt>Pyomakzherchi</rt></ruby></b>**：各氏族的决策机构。氏族间设有更高层级的<b><ruby>大长老会<rt>Lapyomokzhorchi</rt></ruby></b>。

### 头衔演变
- **<b><ruby>长老<rt>Pyomak</rt></ruby></b>**：原指氏族领袖，后逐渐演变为贵族的代称。
- **<b><ruby>萨劼皮奥马克<rt>Sajepyomak</rt></ruby></b>**（世界长老）：咕洛族最高统治者称号。并在并入图斯克帝国后，转而用来称呼帝国皇帝。
- **<b><ruby>居皮奥马克<rt>Jupyomak</rt></ruby></b>**（太阳长老）：帝国时期对咕洛王的专用称呼。

### 宗教与灵语者
咕洛族未形成严密的宗教组织，信仰体系主要通过<b><ruby>灵语者<rt>Johuzay</rt></ruby></b>维系。
- **职能**：通晓<b><ruby>灵语<rt>Johuz</rt></ruby></b>的人被称为灵语者，拥有预言、治疗及沟通灵魂的能力。
- **传承**：他们形成了一个松散的贵族阶层，相关知识仅通过口耳相传，严禁书面记录。

## 经济与文化

### 经济模式
- **畜牧业**：以放牧牦牛、山羊等牲畜为核心，产出毛皮、帐篷材料、衣物及主要食物。
- **农业**：在近河地区开垦有少量耕地，主要种植耐寒的稻谷类作物，作为食物补充。

### 文化特征
- **信仰**：融合了深厚的祖先崇拜与原始宗教信仰。
- **语言**：历史上并无统一的标准官方语言，主要分为沿海方言与内陆方言。现存记录在案的**咕洛语**（详见[[../../50_语言文化/语言/咕洛语|咕洛语]]条目）主要基于南部的沿海方言。
