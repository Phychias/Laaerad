# 政治势力

本目录收录了拉埃拉德世界中核心的政治实体、国家政权及部落联盟。目前的档案主要集中在**图斯克文明圈**及其周边的地缘政治力量。

## 📂 图斯克 (Trusk)

该子目录涵盖了围绕黑蔑特海峡建立的核心文明及其周边互动的政治实体。

| 政治实体 | 类型 | 核心区域 | 描述 |
| :--- | :--- | :--- | :--- |
| **[图斯克帝国](图斯克/图斯克帝国.md)** | 君主专制帝国 | 黑蔑特海峡两岸 | **文明核心与商业枢纽**。<br>建立于三片旧大陆交汇处，扼守海峡天险。以海神信仰为精神支柱，推行“君权神授”；以航运商贸为经济支柱，形成了独特的“官商”阶层。其文化兼具崇尚自由冒险与重视家族荣誉的二元特质。 |
| **[咕洛诸部](图斯克/咕洛诸部.md)** | 部落联盟 $\rightarrow$ 王国 | 帝国东北边境<br>(米斯特那提拉) | **北方的游牧与灵语者**。<br>分布于黑蔑特海峡以东、森林以北的广袤区域。早期为松散的游牧渔猎部落，后由**莫斯瓦里克王朝**完成形式统一，建立了集权的政治体制与“灵语”魔法体系，是帝国北方的重要屏障与兵源地。 |

## 📜 详细档案摘要

### 1. [图斯克帝国](图斯克/图斯克帝国.md)
*   **地缘战略**：连接东西大陆的天然枢纽，控制着诺瓦亚海与新那支海的贸易命脉。
*   **社会结构**：
    *   **民族**：融合了图斯克人、咕洛人、拉普里奥人等多民族。
    *   **语言**：官方通用**黑蔑特方言**（元音开口大、重音靠后），境内并存拉普里奥方言与北部咕洛方言。
*   **宗教信仰**：**海神教**。核心为六主神体系（上三神：织海、藏海、翻海；下三神：涂海、离海、长海），信奉皇帝死后升格为淹神的教义。

### 2. [咕洛诸部](图斯克/咕洛诸部.md)
*   **起源**：帝国东北境的渔猎游牧民族，传说由女神拉姆若血肉所化。
*   **历史进程**：
    *   **莫斯瓦里克王朝**：由**顾卓八顾**利用宗教与底层革命建立，结束了部落混战。
    *   **改革与融合**：在流亡与复辟过程中，吸收了图斯克的装备与制度，同时保留并发展了独特的灵语（Johuz）魔法体系。
*   **文化特征**：崇尚祖先崇拜，社会基础单位为氏族与家族，拥有独特的**咕洛语**（含特殊元音 *ö*）。
