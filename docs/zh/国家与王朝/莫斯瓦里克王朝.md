# <ruby>莫斯瓦里克<rt>Moswarike</rt></ruby>王朝

**莫斯瓦里克王朝**（Moswarike），又称**咕洛王朝**或**马族王朝**，是图斯克历史上由北方[咕洛族](咕洛族.md)建立的大一统王朝。

该王朝由咕洛族第十代大酋长[图录亥刻](图录亥刻.md)建立。它标志着北方游牧文明首次全面征服并统治南方的农耕与海洋文明。莫斯瓦里克王朝以其强大的“灵语”魔法军团、独特的奴隶兵制度以及严酷的情报统治而闻名，虽然其统治手段相对粗糙，但在客观上极大地促进了图斯克帝国南北民族的血统与文化融合。

## 历史沿革

### 崛起与统一

王朝的奠基者[图录亥刻](图录亥刻.md)是咕洛族历史上的雄主。他结束了咕洛草原长期的部落纷争，统一了各部，并被推举为第十代大酋长（萨劼皮奥马克）。 为了积蓄力量，图录亥刻在执政初期采取了极其务实的低姿态外交策略，表面上向当时的帝国霸主称臣纳贡，麻痹对方的警惕，暗中则在北方厉兵秣马，改革军制。

### 南下征服

当时机成熟，图录亥刻率领经过改革的咕洛大军南下。在与南方[巴西卜王朝](巴西卜王朝.md)的关键战役中，咕洛军队展现了压倒性的优势。除了精锐的骑兵外，图录亥刻大规模投入了**灵语者**（法师）部队。利用当时“快照”距离原点较远、魔法潮汐高涨的契机，咕洛人召唤的风暴与雷电击溃了南方联军的防线。 随后，莫斯瓦里克军队攻陷[哈希特](哈希特.md)，入主[黑蔑特地区](docs/zh/概念/黑蔑特.md)，并继续向南推进，征服了包括[图斯克](docs/zh/概念/图斯克.md)以南在内的大部分领土，建立了疆域辽阔的帝国。

### 衰落与终结

莫斯瓦里克王朝的统治后期面临双重危机：

1. **“魔法”衰退**：随着世界位面的移动（快照回归），环境中的灵力潮汐开始消退，王朝赖以维系统治的灵语技术效果大幅减弱，军事优势不再。
    
2. **内部腐败与图斯克化**：进入南方的咕洛贵族迅速迅速腐化，且与保留在北方的保守派产生分裂。 最终，在各地起义军与旧贵族势力的反扑下，这个依靠武力与魔法建立的王朝走向了终结。
    

## 政治与军事体制

### 分封与集权并存

图录亥刻推行了改良的分封制度，将征服的土地分封给战功赫赫的将领和归顺的南方贵族，但在核心区域维持高压控制。

### 奴隶兵制度

为了对抗传统的氏族长老权力，图录亥刻废除了部族武装的垄断地位。他大力保障奴隶的基本权利，并从奴隶中选拔强壮者建立了一支**独立于长老会**的直属常备军。这支奴隶军团只向皇帝本人效忠，拥有极高的战斗力与纪律性，是王朝的立国之本。

### 特务统治：末洛法

王朝设立了名为“末洛法”（Molofa“箭”）的情报与监察机构。该组织直接听命于皇帝，以此监视庞大的官僚体系和地方领主，加强了中央对广阔疆域的恐怖统治。

## 社会与文化

- **粗糙的治理**：作为征服者，莫斯瓦里克王朝在行政管理上显得较为粗放，许多南方的精细农业与商业网络在初期遭到破坏，但在后期逐渐恢复。
    
- **民族融合**：为了巩固统治，王朝鼓励甚至强制推行咕洛人与图斯克人的通婚。这一时期，大量咕洛词汇融入图斯克语，南北方的生活习俗发生了不可逆转的深度融合。
    
- **灵语技术**：这是灵语研究的黄金时代。皇室资助了大量关于魔法、星象与草药的研究，许多关于灵语的经典著作便是在这一时期编纂完成的。
