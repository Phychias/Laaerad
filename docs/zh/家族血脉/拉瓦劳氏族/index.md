# 拉瓦劳氏族

<b><ruby>拉瓦劳<rt>La Valau</rt></ruby></b>氏族是南方[拉普里奥人](../地理风貌/区域/拉普里奥.md)中最具影响力的氏族，亦是图斯克[拉普里奥王朝](../历史长河/朝代更迭/拉普里奥王朝.md)（又名**诺瓦亚王朝**、**拉瓦劳王朝**）的建立者。

详细词条见：[拉瓦劳氏族](拉瓦劳氏族.md)

## 主要人物

-   **氏族奠基者**：[德卓黑一世](../家族血脉/铁苏库/德卓黑一世.md) 通过研习该氏族留下的传送法阵技术，平定了南方叛乱并重塑帝国中央集权。
-   详见主条目：[拉瓦劳氏族](拉瓦劳氏族.md)
