# 拉普里奥语

!!! Abstract "前言"

拉普里奥语是帝国南疆的[拉普里奥](../../10_地理概览/区域/拉普里奥.md)人使用的一种语言，现在已经逐渐式微，许多地区的人开始讲一种图斯克语的拉普里奥方言。

---

## 1 音系 (Phonology)

古拉普里奥语有一种敲击水晶般的听感：清脆、有力、界限分明。

### 1.1 元音 (Vowels)

古拉普里奥语拥有 **5 个基础单元音** 和 **5 个核心双元音**。

#### 1.1.1 单元音 (Monophthongs)

元音在重读和非重读音节中均保持音质稳定，**严禁弱化**。

| **前元音 (Front)** | **中元音 (Central)** | **后元音 (Back)** |           |
| --------------- | ----------------- | -------------- | --------- |
| **闭 (Close)**   | **i** /i/         |                | **u** /u/ |
| **半闭 (Mid)**    | **e** /e/         |                | **o** /o/ |
| **开 (Open)**    |                   | **a** /a/      |           |

#### 1.1.2 双元音 (Diphthongs)

双元音通常源自原始拉埃拉德语长元音的**裂化 (Breaking)**。

| **正字法** | **IPA** | **词源对应 (PL)**    |
| ------- | ------- | ---------------- |
| **au**  | /au/    | $\bar{a} \to au$ |
| **eo**  | /eo/    | $\bar{e} \to eo$ |
| **ie**  | /ie/    | $\bar{i} \to ie$ |
| **oi**  | /oi/    | $\bar{o} \to oi$ |
| **ia**  | /ia/    | 后缀特化             |

### 1.2 辅音 (Consonants)

古拉普里奥语包含 **19 个辅音音位**。其显著特点是塞音系统的**清化倾向**（Devoicing Trend）。

#### 1.2.1 辅音表

| **发音部位 →发音方式 ↓**   | **双唇音(Bilabial)** | **齿龈音(Alveolar)** | **硬腭/后龈(Palatal)** | **软腭音(Velar)** | **声门音(Glottal)** |
| ------------------ | ----------------- | ----------------- | ------------------ | -------------- | ---------------- |
| **塞音 (Stop)**      | **p** /p/, (b)    | **t** /t/, (d)    |                    | **k** /k/, (g) |                  |
| **鼻音 (Nasal)**     | **m** /m/         | **n** /n/         |                    |                |                  |
| **擦音 (Fricative)** | **f** /f/         | **s** /s/         | **x** /ʃ/          |                | **h** /h/        |
| **流音 (Liquid)**    |                   | **l** /l/         |                    |                |                  |
| **颤音 (Trill)**     |                   | **r** /r/         |                    |                |                  |
| **半元音 (Glide)**    | **w** /w/         |                   | **y** /j/          |                |                  |

### 1.3 音节与重音 (Syllable & Prosody)

#### 1.3.1 音节结构

古拉普里奥语允许复杂的音节首（Onset），但保持相对整洁的音节尾（Coda）。

- **最大模版**：$(C)(C)V(V)(C)$
    
- **复辅音簇 (Clusters)**：
    
    - 允许 **“塞音+流音”** 组合出现在词首：**Kl-** (_Klau_), **Pr-** (_Prakit_), **Tr-** (_Trot_), **Kr-** (_Kreat_).
        
    - 允许 **“s+塞音”** 组合：**Sp-** (_Spiom_), **St-**.


这一章节是古拉普里奥语最复杂、最迷人的部分。它是区分“雅言”与“俗语”的分水岭，确立了该语言**“框式结构”**（前缀定性+后缀定格）的屈折语特征。

## 2 词形态 (Morphology)

古拉普里奥语的词汇——尤其是名词——很少裸露出现。它们通常被包裹在一个严密的语法框架中：

$$\mathbf{Prefix} \text{ (前缀: 定性)} + \mathbf{ROOT} \text{ (词根)} + \mathbf{Suffix} \text{ (后缀: 定格)}$$

这种结构赋予了语言极高的稳固性和仪式感。

### 2.1 名词 (Nouns)

#### 2.1.1 性别系统 (Gender System)

古拉普里奥语的性别不基于生物学性别（阴/阳），而是基于**空间感知**与**神学本质**。这是理解该语言世界观的关键。

- **Li 类 (Li-Class / Civic / Proximal)**
    
    - **含义**：俗世的、近处的、人造的、具体的、可控的、社会性的。
        
    - **前缀**：**Li-** (元音前缩合为 **L'**)
        
    - **典型范畴**：人类、职业、工具、建筑、家畜、具体的小物体。
        
- **Na 类 (Na-Class / Natural / Distal)**
    
    - **含义**：自然的、远方的、神圣的、宏大的、不可控的、抽象的。
        
    - **前缀**：**Na-** (元音前缩合为 **N'**)
        
    - **典型范畴**：自然元素（海/火/风）、天体、神灵、抽象概念（爱/死/时间）、大型地理特征。
        

#### 2.1.2 变格系统 (Declension)

为了支持庄重的 **SOV** 语序（即宾语在动词前），必须通过后缀清晰地区分主语和宾语。

| **格位 (Case)**  | **功能**      | **Li 类后缀 (单/复)** | **Na 类后缀 (单/复)** | **听感特征**                |
| -------------- | ----------- | ---------------- | ---------------- | ----------------------- |
| **主格 (Nom)**   | 句法主语        | **-os / -i**     | **-as / -ae**    | **-s** 结尾，清脆有力，表示动作发出者。 |
| **宾格 (Acc)**   | 直接宾语        | **-on / -in**    | **-an / -aen**   | **-n** 结尾，带有延续感，表示承受者。  |
| **与/斜格 (Dat)** | 间接宾语/工具/介词后 | **-om / -im**    | **-am / -aem**   | **-m** 结尾，闭口音，表示静态或终点。  |
| **属格 (Gen)**   | _所有格_       | **-is**          | **-is**          | 通常配合介词 **Ti** 使用。       |

> **变格演示：**
> 
> - **词根**：`Kom` (人 - Li类)
>     
>     - **主语**：_Li-Kom**os**_ (那人...)
>         
>     - **宾语**：_Li-Kom**on**_ (把那人...)
>         
> - **词根**：`Falar` (海 - Na类)
>     
>     - **主语**：_Na-Falar**as**_ (那海...)
>         
>     - **宾语**：_Na-Falar**an**_ (把那海...)
>         

---

### 2.2 形容词 (Adjectives)

形容词在古拉普里奥语中是**从属成分**，它必须“依附”于名词，并在语音上模仿名词的形态。

#### 2.2.1 语序与一致性 (Concord)

1. **位置**：永远置于名词**之后**。
    
2. **规则**：形容词不带前缀，但必须加上与名词**格、数、性**对应的后缀。
    

$$\text{结构：} [\text{冠-名-格}] + [\text{形-格}]$$

**示例：**

- **Li 类修饰**（以 `Sykio` 剑 + `Bord` 直 为例）：
    
    - 主语：**Li-Sykios Bordos** (那把直的剑...)
        
    - 宾语：**Li-Sykion Bordon** (把那把直的剑...)
        
- **Na 类修饰**（以 `Falar` 海 + `Klafias` 宏大 为例）：
    
    - 主语：**Na-Falaras Klafias** (那片宏大的海...)
        
    - 宾语：**Na-Falaran Klafian** (把那片宏大的海...)
        

#### 2.2.2 比较级与最高级

- **比较级**：词根 + **-ior**
    
    - _Klau-ior_ (更大的)
        
- **最高级**：词根 + **-issim**
    
    - _Klau-issim_ (至大的/最大的)
        

---

### 2.3 动词 (Verbs)

动词是句子的**锚点**，位于句末。其形态主要标记**时 (Tense)** 和 **体 (Aspect)**，词干通常以辅音结尾。

#### 2.3.1 时体后缀表

| **范畴**    | **后缀**      | **含义** | **示例 (词根: Kem- 存在)**                                          |
| --------- | ----------- | ------ | ------------------------------------------------------------- |
| **一般现在时** | **-a / -e** | 陈述当前状态 | _Li-Komios **Keme**._ (人存在。)<br>_Na-Falaras **Kema**._ (海存在。) |
| **一般过去时** | **-it**     | 叙述过去事实 | _**Kemit**._ (曾存在。)<br>_注：-it 是最常见的叙事后缀。_                     |
| **一般将来时** | **-niat**   | 预言或计划  | _**Kemniat**._ (将存在。)                                         |
| **进行体**   | **-ko**     | 正在进行   | _**Kemko**._ (正在存在/持续中。)                                      |
| **完成体**   | **-pe**     | 动作已完结  | _**Kempe**._ (已存在过。)                                          |

#### 2.3.2 否定 (Negation)

否定词 **ma** 独立存在，置于动词之前。

- _Li-Komios ma Pakit._ (那人没有杀。)
    

#### 2.3.3 动词名词化

使用后缀 **-da** 将动词转为名词（通常归为 Li 类，作为一种“行为事实”）。

- _Prak-_ (买) $\to$ **Li-Prakda** (购买行为/交易)。
    

---

### 2.4 代词 (Pronouns)

代词保留了古老的形态，不加前缀，但有格的变化。

| **人称**           | **主格 (Nom)** | **宾格 (Acc)** | **属格 (Gen)** | **备注**  |
| ---------------- | ------------ | ------------ | ------------ | ------- |
| **我 (1SG)**      | **Hira**     | **Hiram**    | **Hia**      |         |
| **你 (2SG)**      | **Sot**      | **Soto**     | **Sota**     |         |
| **他 (3SG-Li)**   | **Mif**      | **Mifon**    | **Mifis**    | 指代人/俗世物 |
| **它/祂 (3SG-Na)** | **Maf**      | **Mafan**    | **Mafis**    | 指代神/自然物 |
| **我们 (1PL)**     | **Hiri**     | **Hirim**    | **Hie**      |         |
| **你们 (2PL)**     | **Soti**     | **Sotim**    | **Soe**      |         |

---

### 2.5 虚词与介词 (Particles & Prepositions)

介词前置，要求后面的名词使用**与/斜格 (-om/-am)**。

- **Ti** (之/Of)：表示属格关系。
    
    - _Iklis **ti** Sauh_ (光 之 月)。
        
- **Em** (在/In)：表示方位。
    
    - _**Em** Li-Seream_ (在 房子-里)。
        
- **Tra** (在...之间/Between)：
    
    - _**Tra** L'Iklisam_ (在 光辉-中)。
        
- **Ay** (和/And)：连接词。

## 3. 句法 (Syntax)

### 3.1 基本语序 (Basic Word Order)

古拉普里奥语是严格的 **SOV (Subject-Object-Verb)** 语言。

与图斯克语（SVO）的“线性流动”不同，拉普里奥语的句子结构具有很强的**封闭性**与**悬念感**。动词作为句子的“封印”，必须置于末尾，标志着一个完整意群的结束。

> 句法公式：
> 
> $$\mathbf{S} \text{ (主语-主格)} + \mathbf{O} \text{ (宾语-宾格)} + \mathbf{V} \text{ (动词-时态)}$$

**示例：**

> Li-Komios Li-Sykion Prakit.
> 
> Li-人-主格 Li-剑-宾格 买-过去时
> 
> 中译：那人买了那把剑。

文化隐喻：

这种语序反映了拉普里奥人的思维模式——先陈述参与者（谁和谁），最后才揭示发生了什么（动作）。这在贸易谈判和神学辩论中尤为重要，因为它迫使听众必须耐心地听完整个句子。

---

### 3.2 名词短语 (Noun Phrases)

名词短语内部遵循 **中心语前置 (Head-Initial)** 原则。核心名词位于首位，充当短语的基石；修饰成分（形容词、属格）紧随其后，对其进行雕琢。

#### 3.2.1 形容词修饰

形容词置于名词之后，并严格遵循**格位一致性 (Case Concord)**。

> **结构**：`[冠-名-格]` + `[形-格]`
 
**示例**：

- **Li-Sykios Bordos** (那把直的剑 - 主语态)
    
- **Li-Sykion Bordon** (那把直的剑 - 宾语态)
    

#### 3.2.2 属格结构 (The Genitive)

表示所属关系时，采用介词 **Ti** (之/of) 连接。拥有者通常采用**属格后缀 (-is)**。

> **结构**：`[被拥有者]` + **Ti** + `[拥有者-is]`

**示例**：

> na-iyelas ti sausam
> 
> Na-光辉 之 Na-月-属格
> 
> 中译：月之光 / 月光

---

### 3.3 介词短语 (Prepositional Phrases)

介词（如 _Em_ 在, _Tra_ 之间）置于名词之前。介词后的名词必须使用 **与/斜格 (Dative/Oblique)**，即以 **-m** 结尾。

> **结构**：`Prep` + `[冠-名-斜格(-m)]`

**示例**：

> em asot-fiorom
> 
> 在 时代-古老-斜格
> 
> 中译：在古老的时代

---

### 3.4 句子的扩充 (Sentence Expansion)

当句子包含状语（时间/地点）时，古拉普里奥语倾向于将**状语前置 (Topic-Fronting)**，置于主语之前，作为整个句子的背景框架。

> 扩充公式：
> 
> $$\mathbf{Adv} \text{ (状语)} + \mathbf{S} \text{ (主语)} + \mathbf{O} \text{ (宾语)} + \mathbf{V} \text{ (动词)}$$

**例句分析**：

> **"Em Na-Falaram, Li-Komios Na-Sohusan Spiomit."**
> 
> 1. **em na-falaram**: (状语) 在那片海里 (_Falar-am_ 斜格)。
>     
> 2. **li-komios**: (主语) 那人。
>     
> 3. **na-sohusan**: (宾语) 那神 (_Sohus-an_ 宾格)。
>     
> 4. **Spiomit**: (谓语) 看见了 (_Spiom_ 看 + _it_ 过去时)。
>     
> 
> **中译**：在大海之中，人看见了神。

## 3.5 疑问句 (Interrogatives)

### 3.5.1 是非问句

使用疑问助词 **An** 置于句首，动词保持在句末。

> an li-komios li-sykion prakit?
> 
> [问] 那人 那剑 买了?
> 
> 中译：那人买了那把剑吗？

### 3.5.2 特指问句

疑问代词（谁/什么/哪里）通常移至**句首**，动词依然在句末。

- **makis** (谁-主格)
    
- **maon** (什么-宾格)
    
- **makar** (哪里)
    

> maon li-komios prakit?
> 
> 什么-宾格 那人 买了?
> 
> 中译：那人买了什么？

---

## 3.6 否定与命令 (Negation & Imperative)

### 3.6.1 否定句

否定副词 **nos** 独立存在，置于动词**紧前方**。

> li-komios li-sykion nos prakit.
> 
> 那人 那剑 没有 买。

### 3.6.2 命令句

祈使语气通过动词后缀的变化来实现，通常省略主语。

- **强指令 (Li类)**：后缀 **-tas** (去...!)
    
- **软指令 (Na类)**：后缀 **-as** (愿...)
    

> Prak-tas! (买!) —— 对商人说。
> 
> Sof-as. (去爱吧。) —— 祭司的祝祷。

---

## 3.7 复杂句 (Complex Sentences)

### 3.7.1 并列句

使用连词 **e** (和) 或 **ben** (但是/然而)。

> Li-Komios Kemit, ben Na-Sohusas nos Kemit.
> 
> 人存在，但神不存在。

### 3.7.2 从句 (Relative Clauses)

由于形容词后置，关系从句也置于先行词之后，由关系代词 **Ke** (此/That) 引导。从句内部依然保持 SOV 语序。

> **结构**：`先行词` + **[tse + S + O + V]**

**例句**：

> Li-Komios, [tse na-falan vafit], kemit.
> 
> Li-人, [此(他) Na-海-宾 爱-过去], 存在。
> 
> 中译：那个爱着大海的人，存在着。
