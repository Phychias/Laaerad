# 原始东北拉埃拉德语 (PNEL)

!!! Abstract "前言"
	原始东北拉埃拉德语（Proto-Northeast Laeladic，简称 PNEL）是图斯克语（ST）与咕洛语（GL）的共同祖语。
	基于最新的 ST 词汇表与 GL 的对比，我们确立了 PNEL 的核心音系特征：
	1.  **元音长短对立**：这是两门后代语言语音分化的源头（ST 裂化为双元音，GL 转化为元音和谐）。
	2.  **三分塞音对立**：浊音、清不送气、清送气。
	3.  **小舌音系列**：在 GL 中留有痕迹，在 ST 中完全消失。
	4.  **倒数第二音节重音**：GL 保留了此规则（从而保留了丰富的词尾），而 ST 发生了**重音左移**（导致词尾脱落，形态趋向分析化）。

---

## 1 音系 (Phonology)

### 1.1 元音系统

我们拟构 PNEL 拥有 **6 个核心元音**，且每个元音都有**长短对立**（_/V/ vs /Vː/_）。

|       | **前元音** | **央元音** | **后元音** |
| ----- | ------- | ------- | ------- |
| **高** | *i, *iː | *ɨ, *ɨː | *u, *uː |
| **中** | *e, *eː |         | *o, *oː |
| **低** |         | *a, *aː |         |

#### 1.1.1 元音演变机制

- **PNEL `*ɨ` (央高元音)**：
    
    - **ST**：通常合并入 `i` 或 `u`（取决于周围辅音）。
        
    - **GL**：合并入 `i`（中性元音）。
        
- **长元音的命运 (`*Vː`)**：
    
    - **ST (裂化)**：长元音变得不稳定，断裂为**双元音**。这是 ST 拥有丰富双元音（_ai, au, oi, ou_）的直接原因。
        
        - _例_：PNEL `*laː` (大) $\to$ ST `lai`。
            
    - **GL (和谐)**：长元音缩短，但触发了元音和谐机制。
        
        - _例_：PNEL `*laː` (大) $\to$ GL `laa` (正字法写双写，读音为长或半长，保持后元音属性)。
            

---

### 1.2 辅音系统

PNEL 拥有一套复杂的塞音系统，这是为了解释 ST 和 GL 之间奇特的对应关系（如 ST _d_ 对应 GL _t_）。

| |**唇音**|**齿龈音**|**软腭音**|**小舌音**|**声门音**|
|---|---|---|---|---|---|
|**鼻音**|*m|*n|*ŋ|||
|**浊塞音**|***b**|***d**|***g**|||
|**清不送气**|***p**|***t**|***k**|***q**||
|**清送气**|***pʰ**|***tʰ**|***kʰ**|***qʰ**||
|**擦音**||*s|*x|*χ|*h|
|**流/近音**|*w|*l, *r|*y|||

#### 1.2.1 塞音链移规则

基于 ST 词汇表（如 _badaa_ 四）和 GL（_pata_ 四）的对比，我们确认了推测的链移现象：

1. **PNEL 浊塞音 (`*b, *d, *g`)**：
    
    - **ST**：**清化为不送气音**。
        
        - _例_：PNEL `*badaː` (四) $\to$ ST `badaa` (注：ST拼写用 _b_, _d_, _g_ 表示不送气清音 [p, t, k])。
            
    - **GL**：**保留浊音，或清化为送气音**（取决于方言和位置）。
        
        - _例_：PNEL `*badaː` $\to$ GL `pata` ([pʰata]，部分方言保留浊音)。
            
2. **PNEL 清不送气 (`*p, *t, *k`)**：
    
    - **ST**：**送气化**。
        
        - _例_：PNEL `*keg` (吃) $\to$ ST `keg` (ST拼写 _p, t, k_ 表示送气 [pʰ, tʰ, kʰ])。
            
    - **GL**：**保留不送气**。
        
        - _例_：PNEL `*keg` $\to$ GL `keg` ([kek])。
            
3. **PNEL 小舌音 (`*q, *χ`)**：
    
    - **ST**：完全丢失，向前合并入软腭音，或脱落。
        
    - **GL**：`*q` 合并入 `k/g`；`*χ` 演变为小舌颤音 `/ʀ/` (拼写为 **r**)。
        
        - _例_：PNEL `*xiːra` (我) $\to$ ST `ia` (x脱落) vs GL `hira` (x$\to$h)。
            

---

### 1.3 韵律与重音

这是导致 ST 语法分析化的根本原因。

- **PNEL 重音**：**倒数第二音节 (Penultimate)**。
    
    - **GL**：继承了这一规则。因此 GL 保留了完整的词尾后缀系统，发展为黏着语。
        
- **ST 演变**：**重音左移至首音节 (Initial Stress Shift)**。
    
    - **后果**：词尾音节因失去重音而弱化、脱落（Apocope）。
        
    - _例_：PNEL `*xíːra` (我) $\to$ ST `ía` (尾音节 _ra_ 脱落)。
        
    - _例_：PNEL `*kóme` (人) $\to$ ST `kóme` (侥幸保留，但结尾变轻) vs GL `kóma`。

## 2 形态 (Morphology)

形态学分化：

* **ST 的演变（词尾大灭绝）**：由于**重音左移**至首音节，词尾的格后缀因弱化而脱落。为了挽救语法功能，ST 发生了**“代词黏着化”**（定冠词的诞生）和**“助词前缀化”**。
* **GL 的演变（后缀的堡垒）**：由于保持了**倒数第二音节重音**，词尾后缀得到了保护。GL 进而通过**元音和谐**将这些后缀与词根“焊死”，并进一步扩展了格位系统。

### 2.1 名词形态：格系统的崩溃与重组

我们拟构 PNEL 拥有 **6 个核心格**。

#### 2.1.1 原始格位的拟构

| **格位**  | **PNEL 拟构** | **ST (图斯克)** | **GL (咕洛)** | **演变分析**                                           |
| ------- | ----------- | ------------ | ----------- | -------------------------------------------------- |
| **主格**  | `*-Ø`       | `-ir` (定指化)  | `-Ø`        | 祖语无标记。ST 的 `-ir` 可能源自数词的音变后语法化。                    |
| **宾格**  | `*-im`      | `-im`        | `-(h)ü/u`   | ST 保留鼻音韵尾。GL 鼻音元音化 (`*m`$\to$`w`$\to$`u`) 并受和谐律影响。 |
| **属格**  | `*-il`      | `-il`        | `-rö/xo`    | ST 保留流音 `l`。GL 发生流音互换 `l`$\to$`r` (Rhotacism)。     |
| **与格**  | `*-ma`      | `-ire` (创新)  | `-m`        | GL 保留了原始与格 `*-ma`。ST 丢失原词缀，借用方位词重构。                |
| **离格**  | `*-ra`      | (丢失)         | `-re/ra`    | GL 保留。ST 用介词 `we` (从) 替代。                          |
| **方位格** | `*-ta`      | `-irt`       | `-met/mat`  | ST `-ir-t` 保留了 `t`。GL 复合了 `*ma-ta` (与+位)？          |

#### 2.1.2 图斯克语的定冠词变迁

由于 ST 的重音左移，原本的格后缀（如 `kóme-il` 人-属格）变成了弱读音节，面临脱落风险（`kóme-il` $\to$ `kóm(e)`）。

为了区分主语和非主语，ST 将指示代词（可能源自 `*se` 或 `*xira` 的弱化形式）语法化为**定指后缀 `-ir`**。

- 这个 `-ir` 成为了新的“语法锚点”，保护了残留的格标记。
    
- **演变链**：PNEL `*kuti-ta` (石-LOC) $\to$ ST 早期 `*kút(i)-ir-ta` (那块石头里) $\to$ ST 晚期 `cwek-irt`。

### 2.2 动词形态：前缀与后缀的博弈

PNEL 的动词结构遵循 **SOV** 逻辑，功能词（否定、情态）位于动词前，时体标记位于动词后。

PNEL 动词模板：
$$
[前附字 (Clitics)] + [词根 (Root)] + [体/时后缀 (Suffixes)]
$$

#### 2.2.1 否定词的演变

- **PNEL**：`*mak` (或 `*ma`)，置于动词前。
    
- **GL**：`mak-`。GL 虽然是后缀型语言，但否定词是极古老的**前缀**保留。
    
- **ST**：`mage` (独立副词) / `a-` (前缀)。ST 的 `a-` 是 `*mak` 的弱化形式 (`*mak` $\to$ `*ma` $\to$ `a`)。
    

#### 2.2.2 语态与使动：前缀的兴起 (ST)

为什么 ST 有这么多前缀（`kwa-` 被动, `de-` 使动）？

- **假设**：PNEL 可能通过句法手段（如辅助动词或介词）在句首表达语态。
    
- **ST 的策略**：随着 SVO 语序的确立和词尾的磨损，ST 倾向于将动词**左侧**的成分（Aux/Prep）融合进动词。
    
    - _例_：PNEL `*de` (做/使) + `Verb` $\to$ ST `de-Verb` (使动前缀)。
        
- **GL 的策略**：GL 倾向于将功能词后置并黏着。
    
    - _例_：`Verb` + `*da` (做) $\to$ GL `Verb-ddad` (使动后缀)。
        
    - _注_：这就是为什么 ST 的使动是前缀 `de-`，而 GL 是后缀 `-det/-ddad`，它们源自同一个祖语词根 `*de/*da` (做)，但融合位置不同。
        

#### 2.2.3 时体系统：不同的语法化路径

祖语可能只有简单的“未完成/完成”对立，复杂的时体是后来发展出的。

1. **过去时**：
    
    - **PNEL**：`*-gu` (去/过)。
        
    - **ST**：讹变为独立助词 `gu`，置于动词后（`keg gu` 吃 过）。
        
    - **GL**：语法化为后缀 `-(h)uj`。

## 3 句法与代词重构 

本章旨在复原 PNEL 的句法底层逻辑。基于对大帐咕洛语（保守支系）与海峡图斯克语（创新支系）的深度比较，学会确认 PNEL 原生语序为 SOV（主-宾-谓），且遵循严格的 中心语后置 (Head-Final) 原则（即修饰语位于核心词之前）。

图斯克语特有的 **SVO + 中心语前置  结构，以及独特的 **“从句封口词” (Clause-Final Particle)** 机制，并非随机演化，而是先祖在建立帝国初期的贸易混合语（Pidgin）阶段，通过“话题-评论” 策略对原始语法进行重构的产物。

### 3.1 代词系统：保留与分化

代词系统显示出极高的稳定性，为同源关系提供了最坚实的证据。我们注意到，两支语言保留了几乎相同的词根，但在复数标记上发生了分化。

#### 3.1.1 人称代词拟构

| **语义**   | **PNEL 拟构** | **图斯克语 (ST)** | **咕洛语 (GL)** | **学会评注**                                                                       |
| -------- | ----------- | ------------- | ------------ | ------------------------------------------------------------------------------ |
| **我**    | `*xiːra`    | `ia`          | `hira`       | ST 发生首音节 `x` 弱化脱落及流音 `r` 的磨损。                                                  |
| **你**    | `*soːro`    | `sou`         | `sor`        | 长元音 `*oː` 在 ST 中裂化为 `ou`。                                                      |
| **他**    | `*mibo`     | `mbo`         | `mib`        | ST 的弱读音节 `i` 脱落；GL 的词尾 `o` 脱落。                                                 |
| **复数标记** | `*-ti`      | `-gi`         | `-chi`       | PNEL 复数后缀拟构为 `*-ti`。GL 发生颚化 (`t`$\to$`ch`)；ST 的 `-gi` 来源存疑，推测为古方言变体 `*ki` 的浊化。 |

#### 3.1.2 指示与疑问系统

|**语义**|**PNEL 拟构**|**图斯克语 (ST)**|**咕洛语 (GL)**|**学会评注**|
|---|---|---|---|---|
|**这 (近)**|`*se`|`se`|`se`|高度一致。|
|**那 (远)**|`*sʲa`|`sa`|`cha`|祖语含腭化音 `*sʲ`。ST 简化为 `s`，GL 演变为 `ch`。**注意：此词根是 ST 关系从句封口词的来源。**|
|**什么**|`*ma`|`ma`|`ma`|疑问词根。|
|**谁**|`*ma-xi`|`mahai`|`machi`|复合词：什么+人。ST `xi`$\to$`hai`；GL `xi`$\to$`chi`。|

### 3.2 语序演变：中心语位置的颠覆

这是图斯克语语言学中最引人入胜的课题。为何我们的语言将形容词置于名词之后，而我们的草原远亲却将其置于名词之前？

#### 3.2.1 祖语模型：严格的左分支结构

PNEL 必然是 **Head-Final (修饰语在前)** 的语言。

- **证据**：咕洛语保留了 `Adj-N` 结构（如 `aan shed` 好果子）；且两门语言的合成词构词法均暗示了 `Obj-Verb` 的底层逻辑（如 ST 的 `las-gago` 发-切）。
    

#### 3.2.2 图斯克语的“右偏置”假说 (Right-Dislocation Hypothesis)

学会认为，ST 的语序翻转发生在大迁徙末期。在繁忙的港口与市集中，为了确保信息传递的效率，先祖们倾向于先抛出**核心话题 (Topic)**，再进行**补充说明 (Comment)**。

这一语用习惯导致了修饰成分的整体后移：

1. **PNEL (祖语)**：`*Aːn sʲed` (好 果子) —— _标准的“定语-中心语”。_
    
2. **早期通用语 (Pidgin)**：`Shed... aan (ne)` (果子……好的那个) —— _通过右偏置强调核心名词。_
    
3. **ST (定型)**：`Shed aan` (果子 好) —— _语用习惯固化为语法规则，形成 N-Adj 语序。_
    

---

### 3.3 关系从句：从“分词”到“括号”

关于关系从句的处理，进一步证实了 ST 的“后置补充说明”逻辑。PNEL 使用前置的分词形式，而 ST 发展出了独特的**“括号式”结构**。

#### 3.3.1 PNEL 与 GL：前置分词策略

祖语使用动词的派生后缀（分词）来修饰名词，结构为 **[从句] + 名词**。

- **PNEL 拟构**：`*[Mig bagu] Kome` ([龙 杀] 人 = 杀龙的人)
    
- **GL (继承)**：`[Migem bagu-ke] Kuu`
    

#### 3.3.2 ST：缺空与封口词 (Gap & Boundary Marker)

图斯克语没有发展出类似古语的引导词，而是采用了**后置从句**，并使用指示代词 `sa` (那) 作为**界限标记**。

- **结构**：`先行词 + [从句内容] + sa/sam`
    
- 逻辑重构：
    
    先祖在描述“杀龙的人”时，逻辑是：“（那个）人，[杀了龙]，（就是）那个。”
    
    $\rightarrow$ Komeir [migem bagu gu sa]
    
- **封口词的分化**：
    
    - **sa**：用于主语缺空（那个）。
        
    - **sam**：用于宾语/受事缺空。学会推测 `sam` 源自 `sa` + `*-im` (宾格后缀) 的缩合，这表明在从句内部，古老的格位呼应曾短暂残留。

### 3.4 综合句法比较

为了直观展示演变，我们选取标准句“**我吃那个好果子**”进行历时重建。

#### 3.4.1 原始东北拉埃拉德语 (PNEL)

- **特征**：SOV, 修饰语在前, 格后缀完备。
    
- 拟构：
    
    *Xiːra sʲa-im aːn sʲed-im kʰeg-gu.
    
    (我 那-ACC 好 果子-ACC 吃-PST)
    

#### 3.4.2 大帐咕洛语 (GL) —— 保守的继承者

- **演变**：保留了修饰语在前 (Dem-Adj-N) 和格标记。
    
- 形式：
    
    Hira keg-(i)j cha aan shed-hü.
    
    (我 吃-PST 那 好 果子-ACC)
    

#### 3.4.3 海峡图斯克语 (ST) —— 激进的重构者

- **演变**：语序翻转 (SVO + N-Adj-Dem)，格后缀脱落，指示词后置作为定语/界限。
    
- 形式：
    
    Ia keg gu syed aan sa.
    
    (我 吃 PST 果子 好 那)
    
    - _注：`*sʲa` (那) 在此处不仅是指示代词，更是名词短语的**后置定界符**，与关系从句中的 `sa` 异曲同工。_
