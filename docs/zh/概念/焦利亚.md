# <ruby>焦利亚<rt>jaudziya</rt></ruby>

## 焦利亚王朝

焦利亚王朝是在[莫斯瓦里克王朝](../国家与王朝/莫斯瓦里克王朝.md)末期，“魔法”衰退、皇庭混乱之际，由蛰伏多年的焦利亚军攻陷黑蔑特后建立的。首任君主提卡一世曾试图推进集权化改革，但其子耶奇卡在权力斗争中获胜后，通过分封破坏了其父的愿景，导致改革破产。焦利亚·耶奇卡死后，国家分裂，图斯克再度陷入内乱。
