# 图斯克神话

## 创世纪

混沌初开，上下前后左右不分，清浊是一体，冷热是一体，爱恨是一体。

第一个祂，从漫天的混沌中找到横纵，织造了无边无际的大海。祂以祂的脊椎作经，祂的神经作纬，海水是丝线，也是祂的血液，所以是咸的。

无边无际的大海啊，没有风也没有浪，不分海之上，不分海之下。

从海中来的，又到海中去。一半的海被藏起，滔天的浪涌。

被藏起的又被翻转，这是第一天。
