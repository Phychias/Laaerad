# 巴西卜王朝

!!! warning "词条待补"
    本词条尚未撰写。巴西卜王朝（BC 278 年 — BC 190 年前后，共约 88 年）是[拉普里奥王朝](拉普里奥王朝.md)与[莫斯瓦里克王朝](莫斯瓦里克王朝.md)之间的过渡王朝。其末代皇帝在征讨[咕洛诸部](../../政治势力/图斯克/咕洛诸部.md)途中去世，王朝被巴西卜氏族趁虚取而代之。

    已知要点：

    - **建立方式**：趁[拉普里奥/诺瓦亚王朝](拉普里奥王朝.md)征讨咕洛后方空虚，巴西卜氏族发动叛乱夺取[哈希特城](../../地理风貌/市镇/哈希特城.md)
    - **主要领土**：[图斯克盆地](../../地理风貌/区域/图斯克.md)与[黑蔑特地区](../../地理风貌/区域/黑蔑特.md)
    - **历史意义**：[莫斯瓦里克王朝](莫斯瓦里克王朝.md)南下征服前的短暂过渡期；据[七家共治](大事件/七家共治.md)相关研究，巴西卜集团（特许商团）早在阿贵吉奥时代便垄断了帝国主要水系的航运与贸易，是七家共治时期的重要权势集团之一

    详细历史可参考：[图斯克帝国史](../图斯克帝国史.md) 中"南蛮入侵"与"游牧治国"两节。

## 主要事件

| 年份 | 事件 |
| :--- | :--- |
| BC 278 | 巴西卜氏族攻陷[哈希特城](../../地理风貌/市镇/哈希特城.md)，[拉普里奥王朝](拉普里奥王朝.md)末代"皇帝"战死或病死于北征途中；朝廷被迫走水路退回[诺瓦亚城](../../地理风貌/市镇/诺瓦亚城.md)，巴西卜王朝开启 |
| BC 190 前后 | [莫斯瓦里克](莫斯瓦里克王朝.md)南下入侵，巴西卜王朝覆灭 |
