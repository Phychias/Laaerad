# 拉瓦劳王朝

!!! info "本词条为重定向"
    **拉瓦劳王朝**是[拉普里奥王朝](拉普里奥王朝.md)的别称之一。该王朝得名于其建立者——南方[拉普里奥人](../../地理风貌/区域/拉普里奥.md)中的[拉瓦劳氏族](../../家族血脉/拉瓦劳氏族/拉瓦劳氏族.md)，亦称**诺瓦亚王朝**（取古拉普里奥语"海边"之意）。

    > 详细王朝沿革请参见主条目：[拉普里奥王朝](拉普里奥王朝.md)

## 别名说明

| 别称 | 来源 |
| :--- | :--- |
| **拉普里奥王朝**（主名） | 因王朝统治核心位于[拉普里奥地区](../../地理风貌/区域/拉普里奥.md)而得名 |
| **诺瓦亚王朝** | 取古拉普里奥语 *Nowaja*（"海边"）之音译 |
| **拉瓦劳王朝**（本词条） | 因建立者[拉瓦劳氏族](../../家族血脉/拉瓦劳氏族/拉瓦劳氏族.md)而得名 |
