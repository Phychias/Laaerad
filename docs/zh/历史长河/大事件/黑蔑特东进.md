# 黑蔑特东进

!!! warning "词条待补"
    本词条尚未撰写。"黑蔑特东进"指[黑蔑特第一王朝](朝代更迭/黑蔑特王朝.md)（或更广义的[图斯克](朝代更迭/奎列奥王朝.md)文明）向东北方向扩张、逐步将[咕洛诸部](../../政治势力/图斯克/咕洛诸部.md)与海峡以东地区纳入帝国影响范围的进程。该词条也可能被用于指代后续[铁苏库](朝代更迭/铁苏库王朝.md)与[伯帖斯抓护](朝代更迭/伯帖斯抓护王朝.md)时期向东部延伸的军事行动。

    已知要点：

    - **大致时间**：BC 883 年（[黑蔑特第一王朝](朝代更迭/黑蔑特王朝.md)建立）至 BC 650 年（七家共治前夕）
    - **地理方向**：从[图斯克盆地](../../地理风貌/区域/图斯克.md)向东越过[黑蔑特海峡](../../地理风貌/山河陆海/黑蔑特海峡.md)，推进至[米斯特那提拉](../../地理风貌/区域/米斯特那提拉.md)草原与森林地带
    - **关键节点**：[波拉斯蓬](../../家族血脉/黑蔑特/波拉斯蓬.md)征服克鲁索法、阿贵吉奥摄政时期的对外征伐
    - **历史意义**：为后续[莫斯瓦里克王朝](朝代更迭/莫斯瓦里克王朝.md)的南下征服奠定地缘基础

    详细历史可参考：[黑蔑特王朝](朝代更迭/黑蔑特王朝.md) 与 [图斯克帝国史](../图斯克帝国史.md)。
