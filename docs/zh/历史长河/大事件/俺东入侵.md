# 俺东入侵

!!! warning "词条待补"
    本词条尚未撰写。"俺东入侵"是一个**多次发生的统称性事件**，指[俺东](../../地理风貌/区域/俺东.md)山地居民对[图斯克帝国](../../政治势力/图斯克/图斯克帝国.md)西北边境的反复侵扰。

    已知要点（按时间逆推）：

    - **最早记录**：BC 2800 年左右，[八股纳洛](../../地理风貌/区域/图斯克.md)与[黑蔑特](../../地理风貌/区域/黑蔑特.md)地区首次结成城邦联盟抵御俺东人
    - **持续影响**：[黑蔑特第一王朝](朝代更迭/黑蔑特王朝.md)时期，俺东人是帝国西北边境的主要威胁之一；[马塔肖萨门一世](../../家族血脉/黑蔑特/马塔肖萨门一世.md)与[马塔肖萨门二世](../../家族血脉/黑蔑特/马塔肖萨门二世.md)均曾出兵抵御
    - **民族特征**：以渔猎为生，民风淳朴而彪悍，战士持盾持斧
    - **长期结果**：与[安纽科萨丛林](../../地理风貌/区域/安纽科萨.md)一同构成了帝国西部扩张的天然屏障

    详细历史可参考：[黑蔑特王朝](朝代更迭/黑蔑特王朝.md) 与 [俺东](../../地理风貌/区域/俺东.md) 区域档案。
