# 长夜之乱

!!! warning "词条待补"
    本词条尚未撰写。长夜之乱发生于**BC 48 年**，是[莫斯瓦里克王朝](朝代更迭/莫斯瓦里克王朝.md)末期一次政变，引发了**长达二十年的割据混战**，并直接导致了[焦利亚王朝](朝代更迭/焦利亚王朝.md)与[黑蔑特第二王朝](朝代更迭/黑蔑特王朝.md)的相继兴起。

    已知要点：

    - **发生时间**：BC 48 年
    - **持续时间**：约 20 年
    - **前置背景**：[莫斯瓦里克](朝代更迭/莫斯瓦里克王朝.md)统治后期"魔法"潮汐衰退、皇庭腐败，太后专权
    - **直接后果**：[焦利亚军](朝代更迭/焦利亚王朝.md)围攻[哈希特城](../../地理风貌/市镇/哈希特城.md)，莫斯瓦里克末代皇庭被破；数年后[黑蔑特第二王朝](朝代更迭/黑蔑特王朝.md)复辟

    详细历史可参考：[图斯克帝国史](../图斯克帝国史.md) 中"游牧治国"末尾与"帝国的雏形"两节。
