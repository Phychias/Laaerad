# 大事件

本目录收录了拉埃拉德历史上影响深远的重大事件与转折时刻。这些事件往往跨越单一王朝的兴衰，成为塑造整个东北大陆文明走向的关键节点。

## 重大事件年表

下表按时间顺序列出已收录的重大事件，每条均附有独立词条。

| 事件 | 时间 | 简述 | 主要影响 |
| :--- | :--- | :--- | :--- |
| [黑蔑特东进](黑蔑特东进.md) | BC 883 – BC 520 | 图斯克文明向东北扩张，逐步将米斯特那提拉纳入势力范围 | 为后续莫斯瓦里克南下征服奠定地缘基础 |
| [俺东入侵](俺东入侵.md) | 多次（最早 BC 2800） | 帝国西北部山地渔猎民族反复侵扰边境 | 与安纽科萨丛林一同构成帝国西部扩张的天然屏障 |
| [七家共治](七家共治.md) | BC 650 – BC 598 | 黑蔑特第一王朝晚期，皇权瓦解，七大家族共治元老院 | 中央集权衰落，地方门阀崛起；为卓奥西一世复辟埋下伏笔 |
| [长夜之乱](长夜之乱.md) | BC 48 | 莫斯瓦里克王朝末期的政变，引发约二十年割据混战 | 直接终结莫斯瓦里克王朝，开启焦利亚与黑蔑特第二王朝 |
| [安纽科萨之乱](安纽科萨之乱.md) | 待考证 | 安纽科萨丛林地区的叛乱或军事行动 | 影响帝国南方边境稳定 |

## 时代划分

重大事件通常按以下时期归类：

- **上古时代**（BC 3500 – BC 1700）：以[吾东入侵](俺东入侵.md)的首次记录为代表
- **城邦时期**（BC 1500 – BC 455）：[黑蔑特东进](黑蔑特东进.md)与[七家共治](七家共治.md)
- **前帝国时期**（BC 455 – AD 1）：[长夜之乱](长夜之乱.md)
- **帝国时期**（AD 1 至今）：后续战事与外患记录
