**黑蔑特家族**是[图斯克帝国](../../30_政治势力/图斯克/图斯克帝国.md)历史上生命力最为顽强、影响最为深远的皇室家族。

## 起源

该家族起源于色尔克山脉，在[奎列奥王朝](../../20_历史长河/朝代更迭/奎列奥王朝.md)晚期，凭借哈希特城的商业贸易积累了惊人的财富，通过控制港口与联姻手段逐步渗透权力中心。黑蔑特家族不仅奠定了图斯克帝国的疆域雏形，确立了官僚与行省制度，更在文化上推动了海神信仰的统一与包容。尽管家族内部多次发生惨烈的血亲复仇（如阿贵吉奥时期与共和时期的分裂），但凭借深厚的根基与政治智慧，他们总能在乱世中通过政治妥协或军事政变重回权力巅峰。

黑蔑特家族也是图斯克历史上最早通过控制神权来加强王权的王族，他们利用图斯克已经存在的原始宗教，结合东方的殖民地传来的海神教，建立了一个统一的宗教体系，并从宗教理论上证明了王位的合法性，开启了图斯克权力集中的一个新的时代。

## 特征

黑蔑特家族的人通常有较为瘦削的身形、修长的面容和较高的颧骨，头发多为深色，虹膜有两种典型颜色：深褐色和深灰蓝色。

黑蔑特家族的族徽是一条跃于浪花之上的蓝灰色的鱼，象征着他们家族的起源和家族一贯圆滑的作风。

| **[波拉斯蓬](波拉斯蓬.md)**       | 第一王朝君主    | **帝国最杰出的军事天才**。私生子出身，通过军功在这张列表中赢得席位。他不仅征服了克鲁索法，更建立了图佩罗堡垒，是帝国南进战略的最高执行者。   |
| **[塔莫格](塔莫格.md)**         | 第一王朝中兴者   | **集权大师**。在位四十余年，结束了长期的外戚（维斯尤丰家族）干政。通过扶植官僚与商人阶层，将皇权推向了第一王朝的顶峰。             |
| **[阿贵吉奥](阿贵吉奥.md)**       | 第一王朝摄政/皇帝 | **铁血独裁者**。从控制军队的亲王到实权皇帝，他发动了消耗巨大的“瑞联地绞肉战”。虽然武功赫赫，但其强硬统治也为王朝后期的动荡埋下伏笔。     |
| **[卓奥西一世](卓奥西一世.md)**     | 共和终结者     | **政治天才与复辟者**。在家族分裂、贵族共和的低谷期，通过高超的政治手腕统合家族分支，终结了七家共治的局面，建立了半君主制的“终身执政官”体系。 |
| **[卓奥西二世](卓奥西二世.md)**     | 第一王朝末代君主  | **悲剧的理想主义者**。试图在贵族势力庞大的时代强行集权，最终导致第一王朝在元老院政变中彻底覆灭。                        |
| 盈古霍                       | 第二王朝开国者   | **蛰伏后的复仇者**。利用焦利亚王朝的衰败发动政变，重建黑蔑特王朝。他通过与蛮族联姻的务实外交，为帝国赢得了宝贵的休养生息时间。         |
| 奥勒曼                       | 第二王朝盛世之主  | **“白银时代”缔造者**。在其治下，农业复苏、贸易垄断，图斯克国力达到鼎盛。他是黑蔑特家族在和平时期治理能力的最高代表。             |
| 安塞特                       | 第二王朝末代君主  | **旧时代的余晖**。在瘟疫与内战中即位，通过与铁苏库家族联姻短暂稳定局势，但最终遭到盟友背叛与暗杀，其死标志着第二王朝的终结。          |
| 法坨一世                      | 第三王朝君主    | **昙花一现的复辟者**。在铁苏库王朝之后短暂复辟，但在与权臣和贵族的斗争中失败，死于“日落宴会”。                        |
