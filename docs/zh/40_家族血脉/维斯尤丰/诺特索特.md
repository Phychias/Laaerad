# 诺特索特

!!! wiki "人物档案"
	
    |   |   |
    |---|---|
    |**头衔**|帝国宰相、图尔哈军统帅、弑君者|
    |**活跃年代**|第一王朝中期 (约前 850 - 前 785)|
    |**核心事迹**|暗杀埃布马克特、拥立马佩图马特复位、对抗亲王党|

> 叔叔会夺你皇位，舅舅会帮你守皇位。

诺特索特是[黑蔑特第一王朝](../../20_历史长河/朝代更迭/黑蔑特王朝.md#黑蔑特第一王朝)历史上贵族作为王族的外戚干政的典型。

## 生平

诺特索特将“权臣”二字演绎到了极致。他冷酷、精明，且毫无道德负担，其政治手腕之精准、算计之深远，在当时无人能及。当亲王谒普肯发动政变，废黜合法皇帝马佩图马特并扶植其年幼的侄子埃布马克特作为傀儡时，多数旧臣选择屈服或流亡。然而诺特索特却看到了危机中的机遇——他悄然潜伏，利用过去二十年间在禁卫军与宫廷内侍中精心编织的复杂网络，一步步布局自己的反击。

他的行动冷静而残忍。即便在埃布马克特正式加冕、获得表面上的合法地位之后，诺特索特也未曾动摇。他策划了一场看似寻常的皇家狩猎，却在密林中安排了一场“意外”事故，使年轻的皇帝埃布马克特坠马身亡。随后，他迅速控制宫廷，以武力强行将被软禁的废帝马佩图马特重新扶上王位，而自己则以“帝国摄政与宰相”之名总揽大权。这一连串操作不仅清除了傀儡皇帝，更彻底架空了谒普肯亲王的影响力，使权力重心急剧转移至他自己手中。

掌权之后，诺特索特进一步强化军事控制。他私自调动本该戍卫首都的精锐部队“图尔哈军”，将其长期驻防于佩图罗堡垒一线，与谒普肯残余势力组成的“亲王党”展开长达十数年的军事对峙。在此期间，他多次以皇帝名义发布征兵令与增税法令，实则全部用于巩固个人权力。皇帝马佩图马特在他手中完全沦为象征性的工具，仅存的作用便是在诏令上加盖玉玺——甚至玉玺也由诺特索特的亲信掌管。

在他实际统治帝国的数十年间，图斯克帝国表面仍由马佩图马特统治，实质上却已处于维斯尤丰家族的军事独裁之下。诺特索特建立了一套以个人忠诚为核心的官僚与军事体系，所有关键职位皆由其家族成员或亲信担任。经济上，他垄断了东方商路与矿山开采；司法上，他常绕过律法直接以“军政需要”处置政敌。尽管后世史书常将他标注为“弑君者”，但在其生前，整个帝国无人敢公开挑战他的权威——诺特索特以其毫无温情却高效无比的统治，证明了在混乱的时代，赤裸裸的权力往往比正统名分更具威力。
