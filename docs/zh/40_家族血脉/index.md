# 家族血脉

本目录收录了拉埃拉德世界（特别是图斯克文明圈）中显赫的家族世系。在图斯克的历史长河中，虽然王朝更迭频繁，但那些根深蒂固的门阀世家往往能跨越政权的兴衰，持续影响着历史的走向。

这里记录了从古老的商业贵族到新兴的军事门阀，以及北方草原的征服者氏族。

## 📜 家族与人物志

### 1. [黑蔑特家族](黑蔑特/index.md) (Heimet)
> **家族格言**：浪潮不息

该家族起源于色尔克山脉，凭借商业贸易崛起，将海神信仰确立为国教。
-   **开创与征服**：
    -   **[马塔肖萨门一世](黑蔑特/马塔肖萨门一世.md)**：王朝实际奠基人，推翻旧制，修筑大道。
    -   **[马塔肖萨门二世](黑蔑特/马塔肖萨门二世.md)**：首位黑蔑特皇帝，完成了从豪商到帝王的转变。
    -   **[波拉斯蓬](黑蔑特/波拉斯蓬.md)**：私生子出身的军事天才，征服南方，建立图佩罗堡垒。
-   **极盛与转折**：
    -   **[塔莫格](黑蔑特/塔莫格.md)**：集权大师，建立文官体系对抗外戚。
    -   **[阿贵吉奥](黑蔑特/阿贵吉奥.md)**：铁血摄政王，发动了惨烈的“瑞联地绞肉战”。
-   **黄昏与终结**：
    -   **[卓奥西一世](黑蔑特/卓奥西一世.md)**：终结“七家共治”的政治天才，任终身执政官。
    -   **[卓奥西二世](黑蔑特/卓奥西二世.md)**：试图改革失败的末代君主，死于流放。

### 2. [铁苏库家族](铁苏库/index.md) (Tyesuku)
> **词源**：暖石（象征铁匠与坚韧）

源自西部的军事贵族，以务实、革新和对古代知识的挖掘著称。
-   **[德卓黑一世](铁苏库/德卓黑一世.md)**：**铁苏库第一王朝建立者**。在瘟疫乱世中崛起，利用拉普里奥人的传送法阵遗产平定四方。
-   **[德卓黑三世](铁苏库/德卓黑三世.md)**：**铁苏库第二王朝建立者**。推行单一货币税制，创立皇家学会，确立了真正的帝国集权体制。

### 3. [维斯尤丰家族](维斯尤丰/index.md) (Wisjiopon)
> **家族格言**：根深者叶茂

帝国历史上最强大的门阀，擅长通过联姻、阴谋与政治投机在乱世中保全并壮大自己。
-   **[赛特维斯](维斯尤丰/赛特维斯.md)**：太皇太后，开启了家族长达百年的“外戚干政”时代。
-   **[诺特索特](维斯尤丰/诺特索特.md)**：冷酷的权臣与弑君者，以宰相之名实际统治帝国数十年。
-   **[奥勒曼](维斯尤丰/奥勒曼.md)**：共和时期的领袖，通过与卓奥西的政治交易保留了贵族特权。
-   **[维特莱](维斯尤丰/维特莱.md)**：旧秩序的守护者，策划元老院政变试图换帝，却意外导致了第一王朝的崩塌。

### 4. [莫斯瓦里克氏族](莫斯瓦里克/index.md) (Moswarike)
> **主要成就**：统一咕洛，建立莫斯瓦里克王朝

-   **[图录亥刻](莫斯瓦里克/图录亥刻.md)**：第十代大酋长，建立了特务机构“末洛法”与奴隶兵制度，完成了咕洛诸部的统一并入主黑蔑特。
