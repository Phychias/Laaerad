# 铁苏库·德卓黑三世

**德卓黑三世是图斯克帝国历史上具有划时代意义的君主**，也是**铁苏库第二王朝**的缔造者。他不仅通过武装起义结束了帝国的长期混乱，确立了中央集权的政治架构，更因在位期间对古代遗迹发掘与语言学研究的鼎力支持，被视为帝国文化复兴的关键推动者。

## 👑 政治与军事成就

德卓黑三世作为<b><ruby>铁苏库<rt>Tesuku</rt></ruby></b>家族的领袖，领导起义攻破了旧帝国首都，建立了铁苏库第二王朝。他的统治标志着图斯克地区首次被整合进一个具备真正中央集权性质的政府框架中。

- **确立集权法统**：推行严格的**嫡长子继承制**，并规定领主绝嗣后封地必须收归中央，有效削弱了地方割据势力。
- **差异化行政管理**：
    - 在首都周边设立**黑蔑特亲王领**，由皇室直接管辖；
    - 在边疆地区如**拉普里奥**与**咕洛诸部**，则保留了相对独立的王权或部落制度，体现了灵活的地缘政治智慧。
- **军事重构**：在首都圈实行军权分立以防拥兵自重，同时在中央建立常驻军主导的防务体系，改变了单纯依赖封建征召兵的旧制。

## 📜 文化与学术贡献

德卓黑三世在文化领域的贡献与其政治成就同样显赫。据《[图斯克语](../../50_语言文化/语言/图斯克语.md)》记载，他在“灵语”的启示下，开启了帝国对古代文明的系统性研究。

- **创立皇家学会**：皇帝亲自下令组建了**皇家学会**，并设立专门机构提供资金与人力支持。
- **考古与语言研究**：在他的推动下，学会发掘了包括米斯特那提拉的咕洛语石碑、拉普里奥方尖碑及古图斯克雕像在内的多处遗迹。这些工作为破译**古图斯克语**及**原始拉埃拉德语**奠定了坚实基础。
- **执政影响**：在其执政的十五年间，是帝国学术研究的黄金时期；他去世后，学会经费一度缩减，导致古迹研究放缓。

## 💰 经济改革

面对战乱导致的经济崩溃，德卓黑三世推行了务实且强硬的经济政策，旨在重建帝国的金融秩序。

- **恢复铸币权**：皇庭收回了货币铸造权，结束了货币混乱的局面。
- **推行税制改革**：在核心区域**黑蔑特**推行单一的货币税制并征收商业税；在经济相对落后的**咕洛诸部**及其他地区，则暂时维持物料交换或实物税的形式，体现了因地制宜的治理逻辑。
