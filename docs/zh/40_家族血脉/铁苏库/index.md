# <ruby>铁苏库<rt>Tyesuku</rt></ruby>

**铁苏库**是拉埃拉德早帝国时期一个举足轻重的概念，它既是指代统治图斯克西部的显赫家族——**铁苏库家族**，也是该家族世袭领地与后来建立的**铁苏库小王国**的地理名称。

## 词源与释义

语言学家将**铁苏库**一词追溯至[拉埃拉德原始语](../../50_语言文化/语言/拉埃拉德原始语.md)

- **词根构成**：由 _tyesu_（暖、热）与 _ku_（石）组成。
    
- **原始含义**：“暖石”或“热石”。学界普遍认为，这一称呼最初是对当地盛行的**铁匠**职业的形象描述——象征着铁矿石在炉火中被反复加热与锻打的过程。
    
- **语义演变**：随着当地一支铁匠行会性质的家族逐渐掌握政治权力，该词逐渐从职业/特产描述演变为家族姓氏，进而成为地名。
    

## 铁苏库家族

铁苏库家族是贯穿图斯克历史的顶级贵族门阀，其家族史是图斯克帝国权力更迭的缩影。

### 早期：蛰伏与崛起

在铁苏库韦伯出现之前，历史铭文中关于该家族的记载零星且边缘，多为进京述职的地方小吏。直到**铁苏库韦伯**担任家主，该家族才正式登上历史舞台。在[黑蔑特第二王朝](../../20_历史长河/朝代更迭/黑蔑特王朝.md#第二王朝)时期，韦伯利用王朝内部的动荡，逐步积累了政治资本。

### [铁苏库王朝](../../20_历史长河/朝代更迭/铁苏库王朝.md)

家族的第一次权力巅峰始于**德卓黑一世。

黑蔑特第二王朝的第三位皇帝（即末代王安赛特）没有直系男性子嗣，作为皇帝的外甥（一说为女婿及宰相），德卓黑一世在瘟疫后的乱世中平定四方，最终接管政权，开启了铁苏库第一王朝**。

### 中期：伯帖斯抓护时期的蛰伏

第一王朝并未长久维持，随后帝国陷入了**伯帖斯抓护王朝**的统治时期。尽管德卓黑一世的直系后裔通过联姻融入了伯帖斯抓护皇室，但铁苏库家族的旁系——即**韦伯二世**这一支，并未断绝。他们被分封回祖地**铁苏库城**，在此韬光养晦，积蓄力量。

### [铁苏库第二王朝](../../20_历史长河/朝代更迭/铁苏库王朝.md)
多年后，**德卓黑三世**作为家族领袖，领导了推翻伯帖斯抓护王朝的起义。他成功再度登上帝位，建立了**铁苏库第二王朝**，并确立了图斯克帝国真正意义上的中央集权统治，被后世视为图斯克真正的帝国统治者。

## 铁苏库地区（铁苏库小王国）

**铁苏库地区**，在行政建制上被称为**铁苏库小王国**。

### 历史沿革

- **起源**：最初仅指[图斯克盆地](../../10_地理概览/地点/图斯克.md)南部、克鲁索法河尽头的**铁苏库城**及其周边，这是铁苏库家族在时期获得的最早分封领地。
    
- **扩张**：随着德卓黑一世的南征北战，以及后来铁苏库第二王朝的建立，该区域被整合为一个主要王国，由铁苏库家族的亲王（铁苏库王）世袭管辖。
    
### 地理疆域

该王国位于帝国西部，主要由以下板块组成：

- **铁苏库山脉**：西部的天然屏障。
    
- **格那拉高地**：连接南方的战略高地。
    
- **铁苏库王领**：核心统治区。
    
- **俺东地**：重要的农业与人口区。
    
- **佩尔南地**：位于佩尔南河上游。

### 政治体制

不同于帝国核心区的郡县制，铁苏库王国采取**分封制**。土地主要分封给在统一战争中立下赫赫战功的将领以及铁苏库家族的旧部（如韦伯系的旁支），形成了拱卫帝国西部的军事贵族集团。
