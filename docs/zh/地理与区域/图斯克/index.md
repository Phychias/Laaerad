# 图斯克地区

**图斯克**，作为地理概念，特指拉埃拉德东大陆中部，由**铁苏库山脉**、**古督卡-阿瑟萨山脉**与**安纽科萨雨林边缘**所围合，北临**诺瓦亚海**的巨大盆地及其周边延伸地带。

这片区域因其得天独厚的封闭性与内部的连通性，形成了独特的独立地理单元。它不仅是图斯克文明的发源地，也是连接大陆南北（咕洛与拉普里奥）与东西（内陆与海洋）的天然枢纽。

## 地理四至

图斯克地区的自然边界极其清晰，这种“三面环山，一面朝海”的格局奠定了其易守难攻的地缘优势：

- **北界**：以**北海**南岸线及**黑蔑特海峡**为界。蜿蜒的海岸线提供了通向外部世界的窗口。
    
- **西界**：**铁苏库山脉**。这是一道高耸的褶皱山系，阻挡了来自西部内陆荒漠的热浪与风沙，也是图斯克的主要矿产带。
    
- **南界**：**古督卡-阿瑟萨山脉**。这一古老而险峻的山系构成了图斯克与拉普里奥地区的天然分界线，只有少数隘口可供通行。
    
- **东界**：**佩尔南河下游三角洲**东侧，逐渐过渡至**安纽科萨**湿热雨林的边缘。这里没有明显的山脉阻隔，而是呈现为气候与植被的渐变过渡带。
    

## 地形地貌

图斯克地区的地形结构可概括为“一盆、一河、一岸、两壁”。

### 1. 图斯克中央盆地

区域的主体，是一个巨大的向心状冲积平原。

- **成因**：由古老的佩尔南河及其支流经过数万年的冲积而成。
    
- **特征**：地势平坦低洼，土壤层深厚且呈微碱性，极为肥沃，是整个东大陆人口密度最高、农业最发达的区域之一，被称为“帝国的粮仓”。
    

### 2. 佩尔南水系 

图斯克的母亲河。发源于南部的古督卡山脉积雪融水，自北向南贯穿整个盆地，然后由东向西最终在黑蔑特附近注入诺瓦亚海。

- **水文**：河流流速平缓，水量充沛但季节性变化明显（冬涨夏落）。
    
- **航运**：佩尔南河是图斯克内部的交通大动脉，中下游河段可常年通航平底驳船，将南方的矿石与中部的粮食运往北方港口。
    

### 3. 黑蔑特海岸

北部沿海地带呈现为典型的**沉降式海岸**。

- **特征**：岸线曲折，岬角与海湾交错分布。山脉余脉直接入海，形成了众多天然的深水避风港（如博尔拉索姆港）。
    
- **黑蔑特半岛**：在海岸线西部突出的半岛，扼守着黑蔑特海峡的咽喉，具有极高的战略价值。
    

### 4. 东西高地

- **西部高地**：铁苏库山脉向盆地过渡的丘陵地带，地势阶梯状下降，多台地，是主要的大型牧场和果园分布区。
    
- **南部山麓**：古督卡山脉北坡，地形破碎，多峡谷和激流，是图斯克防御南方入侵的天然屏障（如瑞联隘口）。
    

## 气候特征

图斯克地区属于独特的**诺瓦亚海式气候**（类地中海气候），具有鲜明的“雨热不同期”特征。

- **夏季（旱季）**：受大陆高压控制，炎热干燥，阳光充足。平均气温可达 30°C 以上。这种气候极有利于小麦、大麦的灌浆成熟，以及葡萄、橄榄的糖分积累，但也对水利灌溉提出了极高要求。
    
- **冬季（雨季）**：受来自诺瓦亚海的西风湿润气流影响，温和多雨。气温极少低于 0°C，河流不封冻，利于冬小麦的生长。
    
- **微气候**：
    
    - **西部山区**：气温随海拔降低，降水较多，冬季山顶有积雪。
        
    - **东部边缘**：受安纽科萨雨林水汽影响，夏季相对湿润，常有雷暴。
        

## 自然资源

### 矿产资源

- **黑色金属**：西部铁苏库地区蕴藏着高品位的**磁铁矿**和赤铁矿，且埋藏浅、易开采，是“图斯克钢”闻名于世的物质基础。
    
- **贵金属**：南部与拉普里奥交界的断裂带（瑞联地区）富含**金矿**脉；西部伴生有**银矿**。
    
- **石材**：盆地边缘盛产优质的石灰岩和大理石，不仅是建筑材料，也是烧制水泥和石灰的原料。
    

### 生物资源

- **农业**：适宜种植硬粒小麦、大麦、葡萄、橄榄、无花果。
    
- **林业**：西部和南部山区生长着茂密的**铁杉**和**橡木**，为造船业和冶炼业提供了优质木材和木炭。
    
- **渔业**：诺瓦亚海大陆架宽阔，寒暖流交汇，盛产鳕鱼、金枪鱼及各类贝类，著名的“黑蔑特紫贝”曾是皇室染料的来源。
