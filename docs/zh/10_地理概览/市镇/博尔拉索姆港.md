# 博尔拉索姆港

!!! wiki "博尔拉索姆港"
    | 属性 | 描述 |
    | :--- | :--- |
    | **原名** | Abaj Borlasom |
    | **建立者** | [马塔肖萨门二世](../../40_家族血脉/黑蔑特/马塔肖萨门二世.md) |

博尔拉索姆港，是黑蔑特地区仅次于伊克列港的第二大港，因为大船上常用长绳子(Borlasom)而得名。在黑蔑特地区西南部，主要辐射诺瓦亚海西侧海域。

博尔拉索姆港在黑蔑特第一王朝时期由马塔肖萨门三世主持修建，马塔肖萨门三世对当时的伊克列港嗤之以鼻，评价那里是“下等人的港口”。他在西南部重新修建了博尔拉索姆港，按照他的标准，采用当地的白色大理石修建了港口的建筑。

博尔拉索姆港的在佩尔南河下游的另一条支流，地势北高南低，一侧有山。
