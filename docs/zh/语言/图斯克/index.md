# 图斯克地区
!!! wiki "🌐 图斯克地区语言"
    
    | | |
    | ----------------- | -------- |
    **语系分类** | 东北拉埃拉德语系 |
    **地理分布** | 拉埃拉德大陆东北部，自黑蔑特海峡沿岸至西部山地及诺瓦亚海沿岸 |
    **主要语言** | [[图斯克语]]、[[咕洛语]]、[[图斯克/拉普里奥语]] |
    **共同祖语** | [[拉埃拉德原始语]] |
    **历史节点** | 原始东北拉埃拉德语 |

**图斯克地区语言**，在历史语言学上被归类为**东北拉埃拉德语系**。这组语言分布于拉埃拉德大陆东北部，覆盖了从[黑蔑特](../../地理/图斯克/黑蔑特/index.md)海峡沿岸到西部山地，以及诺瓦亚海沿岸的广阔区域。

该语系的所有语言均起源于灾变期前的[拉埃拉德原始语](拉埃拉德原始语.md)。在漫长的地理隔绝与社会分化过程中，它们演化出了三种截然不同的类型学特征，分别对应了该地区的三个主要文化圈。

---

## 📜 历史演变

图斯克地区语言的演化史通常以**灾变期**为关键分界线，是其所在文明重建历程的缩影。

### 共同起源
在灾变发生前，大陆东北部通行着统一的[[原始拉埃拉德语]]。这是一种具有复杂复辅音、严格的前缀链（遵循否定-被动-使动顺序）以及长短元音对立的古老语言。

### 大分化时期
灾变导致各定居点被地理屏障隔绝，原始语逐渐分化为[[原始东北拉埃拉德语]]，随后迅速分裂为三支：
*   **🏔️ 西部高地分支**：部落于封闭的山区环境中保留了原始语的**黏着**特性，并在快速的口语交流中演化出了**元音和谐律**，最终形成了[[咕洛语]]。
*   **🌊 东部沿海分支**：
    *   **海峡平原**：为适应军事行政与高效沟通的需求，语言发生了剧烈的**分析语化**，复杂词缀脱落为独立助词，形成了[[图斯克语]]。
    *   **诺瓦亚海沿岸**：受海洋贸易与外来文化影响，语言走向了**屈折语**的道路，发展出了独特的元音音变与性系统，形成了[[图斯克/拉普里奥语]]。

## 🗣️ 主要语言

### 🏛️ [[图斯克语]]
!!! wiki "🏛️ 图斯克语"
    
    | | |
    | ----------------- | -------- |
    **语言类型** | 分析语 |
    **主要分布** | 图斯克帝国核心区、黑蔑特及行政中枢 |
    **社会地位** | 帝国官方语言与通用语 |

**图斯克语**以其严谨的逻辑和硬朗的听感著称。它删减了原始语中复杂的喉音，倾向于简单的开音节。在句法上，它确立了独特的**分裂语序**：主句遵循 SVO（主-谓-宾）语序以陈述事实，但在构词与非限定补语中保留了古老的 OV（宾-谓）语序以定义概念。此外，它发展出了一套后置的定指后缀系统（-ir/-wor）。

### ⛺ [[咕洛语]]
!!! wiki "⛺ 咕洛语"
    
    | | |
    | ----------------- | -------- |
    **语言类型** | 黏着语 |
    **主要分布** | 西部山区、俺东地区及游牧部落 |
    **社会地位** | 部落通用语、萨满祭祀语言 |

**咕洛语**保留了最多的原始语特征，听感粗犷而富有节奏。其核心特征是严格的**元音和谐律**，词缀的元音必须随词根变化。它通过在词根后叠加长串后缀来表达复杂的语法意义（如体、时、语气），是典型的黏着语。在语音上，它保留了古老的小舌颤音与软腭擦音。

### ⛵ [[图斯克/拉普里奥语]]
!!! wiki "⛵ 拉普里奥语"
    
    | | |
    | ----------------- | -------- |
    **语言类型** | 屈折语 |
    **主要分布** | 诺瓦亚海沿岸城邦及贸易航线 |
    **社会地位** | 商业贸易语言、文学与歌谣语言 |

**拉普里奥语**是该语系中最具音乐性的语言，拥有丰富的双元音与连读现象。它发展出了基于“俗世/自然”二元对立的**性系统**（Le/La 冠词系统）。其词尾后缀常与词根融合，引发元音裂化或辅音弱化，具有强烈的屈折一致性特征。

## 🌳 祖语与拟构

为阐明三种语言间的亲缘关系，皇家学会的语言学家拟构了它们共同的祖先节点：
*   **[[拉埃拉德原始语]]**：所有拉埃拉德语言的共同祖先，具有高度复杂的形态变化。
*   **[[原始东北拉埃拉德语]]**：[[拉埃拉德原始语]] 在东北地区的直接后裔，是连接上述三种语言的缺失环节。其拟构解释了同源词在后世产生“SVO 与 OV”以及“黏着与分析”等巨大差异的原因。
