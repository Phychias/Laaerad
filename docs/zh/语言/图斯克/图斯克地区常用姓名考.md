# 图斯克地区常用姓名考

图斯克地区的命名文化与其复杂的语言历史、地理分布及社会阶层紧密相关。从海峡沿岸的帝国核心区，到西部山地的咕洛部落，再到诺瓦亚海沿岸的古拉普里奥城邦，不同的文化圈呈现出截然不同的命名逻辑。本条目旨在梳理这些区域的常用姓名及其背后的词源与文化含义。

## 1 图斯克帝国

作为大陆的主导文明，海峡图斯克语 (truskil mielail）的命名规则体现了严谨的阶级划分与分析语特征。贵族与精英阶层倾向于使用结构复杂的复合词，以彰显家族的历史渊源或道德追求；而平民名字则多具描述性，常与自然景物或出生地相关。

### 1.1 贵族姓氏 (naamar)

图斯克贵族的姓氏通常采用古雅的复合词结构，或是保留了拉埃拉德原始语的痕迹。在构词法上，贵族姓氏常采用宾语前置（OV）或修饰语前置的结构，这在以SVO为主的图斯克语中是一种庄重、概念化的体现。

**皇室与军事贵族** 帝国最显赫的姓氏往往与统治权、战争及守护有关。

- **tyesuku**（铁苏库）：`tyesu` “暖”, `ku` “石”，综合起来意为“暖石”，或指向铁苏库家族早期的职业，即铁匠
    
- **moswarike**（莫斯瓦里克）：意为“骑手”（_moswari_ 骑 +  _ke_ 从事……的人），是最早入驻图斯克的咕洛部族。
    
- **turhaike**（图尔海克）：意为“战士”或“尚武者”（_turhai_ 战斗 + _ke_ 者），象征勇武的战斗精神，是极具代表性的武勋世家大姓，是莫斯瓦里克图斯克化的分家之一。
    
- **jaujia**（焦吉亚）：意为“荣耀卫士”（_jaujia_ 褚，即护卫）
    
- **kuinsyec**（奎恩谢克）：意为“黄金剑”或“王之剑”，奎列奥王朝时代流传下来的贵族名
    
- **laigrakou**（莱格拉寇）：意为“大君”或“领袖”（_gai_ 大 + _grakou_ 夫/主），源自部落时代的首领称谓。
    

**神权与古老血统** 部分姓氏源自创世神话或自然崇拜，象征家族拥有古老的祭祀权或通灵能力。

- **heimarit**（海玛利特）：意为“日赐”或“天恩”（_heim_ 日 + _arit_ 给），指家族先祖有受命于天的传说。
    
- **tauswi**（陶斯威）：意为“天灵”（_tau_ 天 + _swi_ 灵），带有萨满或通灵色彩的古老姓氏。
    
- **hambwasia**（汉布瓦西亚）：意为“神话”或“史诗”，通常是负责记录历史的史官家族。
    
- **hetuke**（赫图克）：意为“夜行者”（_hetu_ 夜 + _-ke_ 者），可能与藏海神信仰或守夜职责有关。
    
- **numsib**（努昔卜）：意为“千叶”（_numrea_ 千 + _Sib_ 叶），源自翻海宗的宗教概念，象征繁荣与无穷变数。
    

**行政与财富** 随着帝国商业与官僚体系的发展，许多姓氏体现了财富、领地与行政职能。

- **ruilansom**（瑞兰索姆）：意为“多财者”或“富足”（_ruilan_ 财产、丰饶 + _som_ 形容词化），黑蔑特城周早期的开发者家族之一，在波拉斯蓬麾下进攻南方拉普里奥人矿区，矿区被分封给该家族，并改名为 ruilian，其家族名遂成 ruilansom。
    
- **goiraudun**（戈劳盾）：意为“宏殿”（_goirau_ 殿 + _-dun_ 指大），是古代图斯克的工匠家族之一
    
- **nowdun**（诺顿）：意为“弘海”或，诺瓦亚海沿岸常见姓氏，没有大的宗族，大多是各港口的小城市贵族。
    
- **eratuku**（埃拉图库）：意为“地主”或“领主”（_erat_ 地 + _uku_ 主），图斯克向各地殖民的过程中，由帝国中央派往管理各地的一批官员的职位，后演化成姓氏。
    
- **wisuku**（维苏库）：意为“林主”，类似埃拉图库，但主要领地多为林地。
    
- **guduke**（古督克）：意为“缝补者”或“储藏”（_gudu_ 缝 + _-ke_ 者），源自手工艺者或管理仓库的官员。
    

**品质** 许多贵族以抽象的美德或愿景作为姓氏，标榜家族的道德高度。

- **lousan**（洛萨安）：意为“正道”或“善途”（_Lous_ 道 + _An_ 正），据考也可能是设计马塔肖萨门大道的家族改名而成
    
- **porlasom**（波拉斯索姆）：意为“古直”或“守旧”（_Por_ 旧 + _las_ 直），象征古老、正直与对传统的坚守。
    
- **ilesait**（伊勒赛特）：意为“辉煌”或“光耀”，古老的家族，古代盛产女祭司的家族
    
- **giateis**（吉瓦泰斯）：意为“秘识”或“藏智”（_gia_ 藏 + _teis_ 知），象征掌握不为人知的知识。
    
- **norkosom**（诺科索姆）：意为“多子的”，寄托着希望家族多子嗣的愿望。
    
- **anioksait**（阿尼奥克赛特）：意为“周全”或“轮回”（_aniok_ 转），表达圆满之意。
    
- **ludimart**（鲁迪马特）：意为“乐土”或“欢聚地”（_ludi_ 玩 + _-mart_ 场所），象征愉悦与和谐。
    
- **drodlaisom**（德罗德莱索姆）：意为“深思”（_drod_ 心 + _lais_ 长+*-som* 形容词化后缀），象征性格稳重。
    
- **aritsom**（阿利特索姆）：象征乐善好施与信守承诺。
    
- **aseasait**（阿塞亚赛特）：意为“非凡”，象征与众不同的家族气质。
    
- **lasjodrod**（莱西奥德罗）：意为“阔心”（_laisiok_ 宽 + _drod_ 心），象征包容与大度。
    

**自然图腾** 部分家族以自然界的事物为姓，体现了图斯克人与环境的连接。

- **seitwise**（塞特维斯）：意为“果树”或“李”（_Seit_ 果 + _Wise_ 树），以植物图腾为姓，象征生命力。
    
- **basib**（巴希布）：意为“水草”或“菰”（_Ba_ 水 + _Sib_ 叶），常见于水乡家族。
    
- **barol**（巴罗尔）：意为“深水”或“沉”（_Ba_ 水 + _Rol_ 深），象征深邃沉静。
    
- **laiswise**（莱斯维斯）：意为“高树”或“杨”（_Lais_ 长 + _Wise_ 树），象征挺拔。
    
- **samemgreke**（萨梅姆格雷克）：意为“山民”或“守山人”，象征山地居民的坚韧。
    
- **betusait**（贝图赛特）：意为“寒冷”或“凛”（_Betu_ 冷），多源自俺东北方移民。
    

### 1.2 贵族名字 (Namo)

贵族的名字更注重个人的品质、愿景，常使用动词派生或名词与形容词的组合。

**帝王与领袖气质**

- **laiswi**（莱斯卫）：意为“宏灵”或“大气”（_Lai_ 大 + _Swi_ 灵），帝王之名，象征精神力量的宏大。
    
- **teiskwa**（泰斯夸）：意为“闻名”或“被知者”（_Teis_ 知 + _-kwa_ 被动），希望孩子名扬天下。
    
- **yingukho**（因古霍）：意为“必胜”（_Ying_ 必 + _Gukho_ 胜），表达对胜利的绝对信心。
    
- **oleman**（奥勒曼）：意为“善果”或“成就”（_Olem_ 结果 + _An_ 好），希望人生有好的结局。
    

**尚武与行动力**

    
- **kansoi**（坎索伊）：意为“快剑”或“瞬斩”（_Kan_ 剑 + _-soi_ 急速），希望孩子剑术超群。
    
- **jeposoi**（耶普索伊）：意为“疾行”或“冲锋”（_Jepo_ 走 + _-soi_ 急速），象征行动力与迅捷。
    
- **jaujyaan**（焦加安）：意为“忠卫”（_Jaujya_ 卫士 + _An_ 好），寓意成为优秀的守护者。
    
- **guit**（贵特）：意为“用石头”，来自古老的投石兵。
    
- **gudun**（古盾）：意为“巨石”（_Gu_ 石 + _-dun_ 指大），象征性格坚毅。
    
- **wetlai**（维特莱）：意为“大风”或“狂风”，象征行事雷厉风行。
	
- **wetpo**（韦伯）：意为“风化”，象征行事像风一样迅捷，但较前一个名字轻盈。

**智慧、精神与美德**

- **Drodaucy**（德罗道奇）：意为“静心”（_Drod_ 心 + _Aucy_ 静），寓意性格沉稳、处变不惊。
    
- **Ningswi**（宁斯威）：意为“能灵”或“灵能”，象征拥有强大的精神潜能。
    
- **Tyaan**（提亚安）：意为“善愿”或“好意”（_Tyaa_ 意愿 + _An_ 好），象征心存善念。
    
- **Azozi**（阿佐齐）：意为“无畏”（_A-_ 无 + _Zozi_ 怕），否定派生名，希望孩子勇敢。
    
- **Porteis**（波特伊斯）：意为“先知”或“预见”，常用于学者或祭司。
    
- **Porteiske**（波特伊斯克）：意为“大巫”或“预言家”，指使预言成真的人。
    
- **Soryaz**（索里亚兹）：意为“织造者”（_Sori_ 织 + _-yaz_ 态），象征具有创造力，能构建秩序。
    
- **Troleug**（特罗列格）：意为“燧人”或“造火”，象征智慧与开创。
    
- **Dayaud**（德奥德）：意为“救赎”或“续命”（_De-_ 使 + _A-ud_ 不死），递归派生词，常用于医生。
    
- **Ansait**（安赛特）：意为“安康”或“平安”，最朴素美好的祝福。
    

**自然与美貌**

- **wespan**（韦斯帕安）：意为“绝色”或“亮彩”（_Wespa_ 色 + _an_ 好的、美的），形容容貌出众。
    
- **Sniowan**（斯尼奥万）：意为“悦目”或“俊美”，形容令人赏心悦目。
    
- **Heimbor**（海姆博）：意为“旭日”（_Heim_ 日 + _-bor_ 起始），象征新生与希望。
    
- **Rigujiu**（里古久）：意为“白雪”，象征品行高洁。
    
- **Bafara**（巴法拉）：意为“流水”，象征生命顺遂。
    
- **Baik**（拜克）：意为“似水”或“柔顺”，性格温润。
    
- **Weten**（韦腾）：意为“风化”或“随风”，象征自由洒脱。
    

## 2 咕洛部落

咕洛语属于黏着语，其姓名风格粗犷，常与猛兽、自然力相关。

**氏族名**

- **Khenxed**（肯赫德）：意为“黑狼”，勇猛好战的部族。
    
- **Olaleuk**（奥拉列克）：意为“火头”或“炎首”，崇拜火神的部族。
    
- **Mörikesep**（莫里克瑟普）：意为“高马”，善于骑射的部族，莫斯瓦里克分家（语音流变）。
    
- **Rigurchi**（里古尔奇）：意为“雪众”或“雪族”，居住在极北冰原的部族。
    

**个人名**

- **Vesnö-sar**（维斯诺萨）：意为“猎手”，职业专家名。
    
- **Bordas-kim**（博尔达斯基姆）：意为“行直者”或“正直”。
    
- **Gok-ola**（格克奥拉）：意为“铁头”或“天首”。
    
- **Joluk-xar**（乔鲁克哈）：意为“暴怒者”。
    
- **Ila-sora**（伊拉索拉）：意为“多光”或“璀璨”。
    
- **Taamek**（塔梅克）：意为“天客”，也作为问候语“你好”。
    

## 3 古拉普里奥城邦

古拉普里奥语受海洋贸易影响，具有流利的双元音和屈折语特征，并发展出了基于“俗世/自然”的性别冠词系统（Le/La）。

**家族/商会名** 通常使用 **La**（远性/自然性）开头的词，表示家族源远流长。

- **La Valau**（拉·瓦劳）：意为“海家族”，垄断航运的巨头。
    
- **La Kuoju**（拉·库尤）：意为“银家族”，银行家或银矿主。
    
- **La Hura**（拉·胡拉）：意为“时家族”，古老的钟表匠或预言家。
    
- **La Sedia**（拉·塞迪亚）：意为“森家族”，木材商人。
    

**个人名** 男性或世俗职业多用 **Le** 类（近性），女性或神职多用 **La** 类（远性）。

- **Arino**（阿里诺）：Le类名，意为“商人”或“贾”。
    
- **Lauro**（劳罗）：Le类名，意为“伟岸者”。
    
- **Foko**（福科）：Le类名，意为“飞翔者”，常指信使。
    
- **Vala**（瓦拉）：La类名，意为“海女”。
    
- **Maura**（莫拉）：La类名，意为“云”或“雾”。
    
- **Leuga**（列加）：La类名，意为“烈火”，指热情或红发的女性。
