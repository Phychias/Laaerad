# <ruby>波拉斯蓬<rt>Porlaspon</rt></ruby>

> 但我对图斯克是没有亏欠的。

<ruby>波拉斯蓬<rt>Porlaspon</rt></ruby>是图斯克历史上一位极具传奇色彩的军事统帅与征服者。作为马塔肖萨门三世的私生子，他早年在宫廷的阴影与歧视中成长，这塑造了他冷酷而坚韧的性格，也驱使他通过赫赫战功来赢得认可与权力。

## 生平

* **南方的征服者** ：波拉斯蓬对哈希特宫廷内的权谋斗争感到厌倦，转而将雄心投向南方未开化的土地。他亲自率领军队三次强渡克鲁索法河，最终彻底击溃了盘踞在该地区的土著部落联盟。
*   **图佩罗的建立** ：为了巩固征服成果，他在克鲁索法河中游建立了一座军事重镇——**图佩罗**（意为“胜利”）。这座城市的设计理念与商业中心哈希特截然不同，它完全是一座为战争打造的堡垒，其街道布局处处着眼于巷战防御，此后这里便成为波拉斯蓬统治与征战的核心基地。
*   **黄金与诅咒** ：波拉斯蓬进一步拓展了图斯克在诺瓦亚东岸的势力范围，建立了稳定的殖民地与军事据点。其最重要的行动是征服了瑞联地，并在该地发现了储量巨大的金矿。源源不断运回哈希特的黄金，极大地巩固了他的政治地位，暂时平息了朝廷中对其出身背景的非议。
