# 黑蔑特家族

**黑蔑特家族**是[图斯克](../../../en/概念/图斯克.md)历史上反复出现的强大贵族家族，其兴起与商业贸易的繁荣紧密相关。在奎列奥王朝晚期，哈希特城凭借优越的港口条件成为重要的商业中心，为黑蔑特家族积累了雄厚的经济实力。
