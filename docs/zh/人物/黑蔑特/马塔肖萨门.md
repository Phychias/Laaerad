# <ruby>马塔肖萨门<rt>Matasioksamem</rt></ruby>

**马塔肖萨门**（Matasioksamem）是古图斯克语中的一个专有名词。虽然在中古图斯克语时期已较为罕见，但在古代常被用作人名或氏族名。此外，该词在现代图斯克语境中，也常指代著名的[马塔肖萨门大道](https://www.google.com/search?q=../%E5%8C%BA%E5%9F%9F/%E9%BB%91%E8%94%91%E7%89%B9/%E9%A9%AC%E5%A1%94%E8%82%96%E8%90%A8%E9%97%A8%E5%A4%A7%E9%81%93.md)。

## 词源与释义

语言学家推测，“Matasioksamem”一词处于东北[拉埃拉德语](https://www.google.com/search?q=../%E6%A6%82%E5%BF%B5/%E6%8B%89%E5%9F%83%E6%8B%89%E5%BE%B7.md)向古图斯克语演变的过渡阶段，其词义构建包含两个部分：

1. 词头 mata-：
    
    被认为是一种方言性的叙述前缀。它可能近似于原始语中的后缀 *-mat*，但在演变过程中，该语素的位置发生了倒置，移动到了词首。
    
2. 词根 sioksamem：
    
    意为“宽阔的山”或“连绵的山峦”。
    

综上所述，该名字的字面含义大意为**“住在山峦里的人”**或**“从山里来的人”**。

## 语法变格

当“马塔肖萨门”用于指代地理设施（如道路）时，通常需要遵循特定的语法规则：

- **定指后缀**：一般会加上 `*-ir*` 后缀作为修饰。
    
- **名词组合**：常与 `*lous*`（道路）或 `*dzal lous*`（大道）连用。
    
    - 例：_Matasioksamem-ir dzal lous_（马塔肖萨门大道）。
        

## 历史考据

根据词源学的含义（山里来的人），历史学家提出了关于[黑蔑特](https://www.google.com/search?q=../%E4%BA%BA%E7%89%A9/%E9%BB%91%E8%94%91%E7%89%B9.md)家族起源的假说：

- **地缘起源**：马塔肖萨门父子可能起源于**色尔克山**（Selk Mountains），这是距离黑蔑特地区最近的山脉，符合“山民”的描述。
    
- **姓氏讹传**：鉴于该词的描述性性质，“马塔肖萨门”极有可能最初并非这几位君主的本名（Personal Name），而是他们的**姓氏**或**氏族名**（Surname/Clan Name）。然而，在漫长的历史流变与口述中，他们的真实名字已经失传，后世遂将这一氏族名讹传为了他们的名字，而将他们最终掌控的核心地区[黑蔑特](../../地理与区域/图斯克/黑蔑特/index.md)作为了他们的姓氏。
