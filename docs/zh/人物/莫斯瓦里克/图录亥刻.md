# <ruby>图录亥刻<rt>Turhaike</rt></ruby>

**图录亥刻**是[莫斯瓦里克氏族](index.md)的第十代大酋长，以其卓越的政治与军事整合能力著称。他成功统一了[咕洛诸部](../../国家与王朝/咕洛诸部/index.md)（至少在形式上），建立起一支高度团结且战斗力强的军队。在其统治期间，推行了包括分封制度、奴隶权利保障以及奴隶兵制度在内的一系列改革，并策略性地向黑蔑特中央暗中称臣，以降低对方的警觉。为强化统治，他创建了情报组织“末洛法组织”，并设立了独立于传统长老会的直属军队。在对外战争中，图录亥刻借助灵语者的力量击败巴西卜，进而入主黑蔑特，并将图斯克盆地以南的广阔地域纳入其控制范围。其名“图录亥刻”来源于图斯克语，意为“好斗的”，是对他在战场上英勇表现的称谓。
