 如果说黑蔑特家族是图斯克帝国的骨架，那么维斯尤丰家族便是附着其上的血肉与神经。他们不追求在台前挥舞权杖，而是热衷于在幕后决定权杖的归属。在图斯克历史上的许多王朝，王权的阴影下四处都是他们的触角。
