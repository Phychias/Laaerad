# 哈希特城

**哈希特城**是哈希特地区的政治、经济和文化中心。该城现坐落于色尔克山南侧，毗邻黑蔑特湾，是帝国版图中的关键枢纽。

## 历史沿革

城市最早建立于旧佩尔南河沿岸。后因遭遇严重洪水灾害，原址被毁，随后迁址至地势更为稳固的色尔克山南麓进行重建。

## 马塔肖萨门改革

在**马塔肖萨门改革**时期，当局修建了连接哈希特城与[图斯克盆地](../index.md)的[马塔肖萨门大道](马塔肖萨门大道.md)。这一工程极大地便利了物资运输与信息的流通，进一步巩固了黑蔑特作为帝国核心的地位。
