# 图斯克语语法概要

## 语序

图斯克语是典型的话题优先语言 (Topic-Prominent)。
- 话题通常位于句首
- 核心名词在前，修饰语后置
- 中心语前置，定语后置

## 名词短语

### 中心语前置 + 定语后置

```
[名词] + [形容词/修饰语]

例: kenir lai     (狗 + 大)   → "那只大狗"
    baaja dziapk  (海 + 碧蓝) → "碧蓝的大海"
```

### 领属结构

```
[被拥有者] + [拥有者-属格]

例: syecir tesukuil  (剑 + 铁苏库的) → "铁苏库的剑"
```

### 格标记

| 后缀 | 格 | 用法 |
|------|-----|------|
| -ir | 主格 (NOM) | 主语，定指 |
| -im | 宾格 (ACC) | 宾语 |
| -il | 属格 (GEN) | 领属 |
| -irt | 从格 (ABL) | 起点 |
| -wor | 主格复数 | 复数主语 |
| -il | 属格 | 领属关系 |

## 动词短语

### 时态系统

| 时态 | 标记 | 位置 |
|------|------|------|
| 过去时 | gu | 动词后 (独立词) |
| 现在时 | (无标记) | 词典形 |
| 未来时 | -in | 动词后 (黏着后缀) |

### 例句

```
例1: heimir deciap gu.
     太阳=定指 升起 过去时
     "太阳升起了"

例2: betumartirt nwi aguisib wise ar.
     寒带地区=从格 复数非定指 针叶的 树 存在
     "在寒带有针叶林"

例3: kenir lai japem keg gu.
     狗=定指 大 肉=宾格 吃 过去时
     "那只大狗吃了肉"
```
