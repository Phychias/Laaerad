# 图斯克语辅音

| 字母 | IPA | 说明 |
|------|-----|------|
| m | /m/ | 鼻音 |
| n | /n/ | 鼻音 |
| ng | /ŋ/ | 软腭鼻音 |
| b | /p/ | 不送气清塞音 (注1) |
| d | /t/ | 不送气清塞音 |
| g | /k/ | 不送气清塞音 |
| p | /pʰ/ | 送气清塞音 |
| t | /tʰ/ | 送气清塞音 |
| k | /kʰ/ | 送气清塞音 |
| h | /h/ | 声门擦音 |
| s | /s/, /ɕ/ | 擦音 (在/i/前为龈颚音) |
| z | /z/, /ʑ/ | 浊擦音 |
| j | /ts/, /tɕ/ | 不送气塞擦音 |
| c | /tsʰ/, /tɕʰ/ | 送气塞擦音 |
| l | /l/ | 边音 |
| r | /r/ | 颤音 |
| w | /w/ | 唇软腭近音 |
| y | /j/ | 硬腭近音 |

> 注1: 图斯克语缺乏浊塞音，b实际发音为/p/，但与送气的p构成对立。
