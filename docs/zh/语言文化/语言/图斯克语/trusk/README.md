## 快速入门

### 音系
- 5个单元音: a, e, i, o, u
- 2个鼻化元音: aa, oo
- 7个双元音: ai, oi, au, ia, iu, ui, ua, ue
- 18个辅音

### 语法特点
- 话题优先语言
- 中心语前置，定语后置
- 高度黏着：前缀 + 词根 + 后缀
- 三层后缀系统：S1(增价) → S2(方式) → S3(时态)

### 核心词缀

| 类型 | 词缀 | 功能 |
|------|------|------|
| P1 | a- | 否定 |
| P2 | kwa- | 被动 |
| P2 | zi- | 反身 |
| P2 | ma- | 交互 |
| S1 | -de | 使动 |
| S1 | -rit | 受益 |
| S1 | -sek | 强迫 |
| S2 | -morg | 彻底 |
| S2 | -soi | 急速 |

| S2 | -mau | 佯装 |
