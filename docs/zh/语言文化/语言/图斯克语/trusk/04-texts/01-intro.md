# 图斯克语示例文本

## 例句集

### 基本句型

```
1. heimir deciap gu.
   太阳=定指 升起 过去时
   "太阳升起了"

2. kenir lai japem keg gu.
   狗=定指 大 肉=宾格 吃 过去时
   "那只大狗吃了肉"

3. betumartirt nwi aguisib wise ar.
   寒带地区=从格 复数非定指 针叶的 树 存在
   "在寒带有针叶林"
```

### 话题优先

```
4. Japem, kenir lai keg gu.
   肉=宾格 狗=定指 大 吃 过去时
   "（至于）那块肉，大狗吃了它"
   → 宾语话题化
```

### 动词变形

```
5. okoir kegeg.
   鸟=定指 吃~REP
   "那只鸟一直吃"
   → 反复体

6. okoir kegegin.
   鸟=定指 吃~REP=未来
   "那只鸟将会一直吃"
   → 反复体 + 未来时

7. ia i koem kegegdein.
   我 一 鸟=宾格 吃~REP-使动-未来
   "我将一直喂那只鸟"
   → 反复体 + 使动 + 未来时
```

### 复杂动词结构

```
8. nemludesoin
   转-使动-急速-未来
   "将要快速旋转（某物）"

9. kahya mapaubau
   海鸥 交互-模仿
   "海鸥们在互相模仿"

10. sywas akwadisyonau
    序列 否定-被动-引导
    "该序列未被引导（处于无序/失控状态）"
```

### 静态/存在句

```
11. ia gamaide mboem
    我 懂-使动 他=NOM.ACC
    "我使他明白/我给他讲解"

12. kizy sisicmau
    鸟类 发疯-佯装
    "那鸟在装疯"
```

## 小对话

### 日常对话

```
A: Heimir deciap gu.
   太阳 升起 过去时
   "太阳升起了"

B: E, ia gamai.
   是 我 懂
   "是的，我知道了"

A: Okoir kegeg.
   鸟 吃~REP
   "鸟在吃东西"

B: Ma, okoir aucibor gu.
   不 鸟 完成 过去时
   "不，鸟已经吃完了"
```
